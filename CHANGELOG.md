## [0.1.0] - 2026-03-22

### Added
- Initial release of the module database
- 17 published modules covering elementary through high school mathematics:
  - Elementary: ELM-001 through ELM-005
  - High school algebra: ALG-001, ALG-002
  - Calculus: CALC-001, CALC-002
  - Functions: FUNC-001, FUNC-002
  - Geometry: GEO-001
  - Statistics: STAT-001
  - Trigonometry: TRIG-001
  - Vectors: VEC-001
- `math/MODULE_SCHEMA.md` defining the module format specification
- `CURRICULUM_SCOPE.md` listing all 101 planned modules with prerequisites
- `CONTRIBUTING.md` with sourcing and formatting guidelines
