## 制限事項について

本モジュール（司書）の役割は、「実際のウェブコンテンツをインターネットから収集して保存すること」のみです。

- **内容の要約・執筆**:これは Editor の役割であり、司書は行いません
- **教材執筆**: これも Editor の領域です
- **収集のみ**: Web 検索から Wikipedia、DDG 上位などを取得し、`references/` に配置することが唯一の任務です

## 次のステップ

もし収集したい場合、以下の作業を行ってから再度検索・収集してください：

1. `web_search.collect_references()` の呼び出し
2. 日本語 Wikipedia 記事（CC BY-SA 4.0）の優先取得
3. DuckDuckGo「数学 正負の数 絶対値 解説 教材」検索
4. 上位ページの収集と保存機能

資料が収集でき次第、INDEX.md に整理し、Editor へ渡します。現在はネットワークエラーまたは収集漏れのため、空のINDEX.mdをご提供いたします。
