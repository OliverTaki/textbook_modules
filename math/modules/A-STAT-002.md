## 概念説明
折れ線グラフとは、データが**時間や順序**といった連続的な変数の経過とともにどのように変化していくかを示すためによく用いられるグラフの一種です。棒グラフが各時点やカテゴリごとのまとまった量を比較するのに対し、折れ線グラフは点と点を線で結ぶことで、その変化の流れ（傾向）を視覚的に捉えることを主目的としています (参考: [1])。学習する上で最も重要なポイントは、この「繋がり」にあります。線で結ばれている箇所は、データの飛び石ではなく、時間の経過に伴って自然に変化していく過程、つまり連続的な変化を表現していると理解することが大切です。

グラフの構成要素としては、横軸（$x$軸または独立変数）が時間の経過（年月日や年数など）を示すことが多く、縦軸（$y$軸または従属変数）が測定された値（気温、株価、物体の移動距離など）を示します。プロットするデータポイントの大きさ自体が個々の測定値を示しますが、そのポイント間を結ぶ線が、その間の**推移の勢いやパターン**を読み取らせる役割を果たします。

例えば、ある地域の過去10年間の平均気温の変化を折れ線グラフにした場合、線が急激に上向きになっている区間は「気温が急激に上昇した期間」、緩やかに下降している区間は「気温が徐々に低下した傾向」と読み取ることができます (参考: [7])。このグラフから読み取れるのは、あくまで「傾向」であり、特定の時点における正確な値そのものではないに注意が必要です。また、折れ線グラフを使用する際は、基本的にデータが連続的に観測されている、あるいは連続的なプロセスを描いている場合に適しており、単にA地点とB地点の値を同時に比較したいだけであれば棒グラフの方が適している場合があるため、データの性質を見極めることが統計的な解釈力の基礎となります。

## 例題

**【例題1：気温の変化の読み取り】**
ある都市の過去5日間の最高気温（℃）のデータが以下のように記録された。これを折れ線グラフにしたとき、どの期間に最も急激に気温が低下したか問い、その変化の大きさを求めなさい。

| 日付 | 最高気温（℃） |
| :--- | :--- |
| 1日目 | 25 |
| 2日目 | 27 |
| 3日目 | 23 |
| 4日目 | 19 |
| 5日目 | 20 |

**【解答ステップ】**
1.  **データの確認:** グラフ化すべきデータは、横軸に「日数」、縦軸に「最高気温」をとる。
2.  **変化の計算:** 気温の「変化の大きさ」は、前日からの差分 $|A - B|$ として計算する。
    *   1日目→2日目: $|27 - 25| = 2$ ℃ (上昇)
    *   2日目→3日目: $|23 - 27| = 4$ ℃ (下降)
    *   3日目→4日目: $|19 - 23| = 4$ ℃
    *   4. 4→5日目: $|20 - 19| = 1$ ℃
3. **結論:** 差が最も大きかったのは、2日目から3日目にかけての低下（4°C）であり、これは2回発生しています。ただし、最初の急な低下（2→3日目、4°C）が最も急な変化点であると判断できます。

**【解説】**
急な変化を見るには、隣接するデータポイント間の差（絶対値）を計算し、その最大値を見つける必要があります。

**例題2：棒グラフと折れ線グラフの使い分け**
ある町の毎月の売上データが以下の通り集計された。
1月売上：50万円
2月売上：70万円
3月売上：30万円
この売上動向をグラフ化すると、売上の「推移（時間の経過に伴う変化）」がわかるグラフと、「各月の絶対的な大きさ」がわかるグラフのどちらに適しているか答えよ。

**【解説】**
売上の「推移」を捉えることは最重要課題です。時間の経過に伴う変化の繋がりを重視するため、折れ線グラフが最も適しています。棒グラフは、各月の売上の絶対的な差の比較には優れていますが、時間的な連続性を示す点では折れ線グラフに劣ります。

## 練習問題

あなたは市場調査会社に勤めています。ある地域の過去数年間の平均気温の変化をグラフ化し、クライアントに報告するよう依頼されました。以下のデータに基づき、最も適切なグラフの種類を答え、その理由を簡潔に記述しなさい。

**データ（平均気温：℃）**
| 年 | 気温 |
| :---: | :---: |
| 2018年 | 14.2 |
| 2019年 | 14.5 |
| 2020年 | 14.8 |
| 2021年 | 15.1 |
| 2022年 | 15.3 |
| 2023年 | 15.0 |

**設問1：最も適したグラフの種類を答えなさい。**
**設問2：そのグラフを選んだ理由を、データ活用の観点から記述しなさい。**
