# 三角形の合同条件

## 1. 単元の解説

三角形の合同条件を理解することは、平面幾何学において最も基礎となる重要な単元の一つです。本単元の目標は、三角形が合同である十分必要条件を正しく理解し、異なる与えられた条件がどの合同条件を満たすのかを特定することにあります。

三角形の合同とは、2 つの三角形を重なり合わせ合わせたときに、完全に一致（ぴったり合う）することです。具体的には、各辺の長さ（長さ）と各角の大きさがすべて等しいことになります。これに対し、相似とは、形は同じくても大きさ（縮尺）の比例が異なる状態を指します。相似条件（辺の比が等しい、角度が 2 つ等しい）は、合同条件とは異なります。特に、合同条件を満たすことは合同を意味しますが、相似条件を満たすだけで合同であるとは限りません。

合同条件は、通常「SSS、SAS、ASA、AAS」の 4 つで成り立つことが知られていますが、SSA（2 辺 1 角で、角が挟んでいない）は合同条件にはなりません。なぜ SSa 条件が不十分なのかを理解するために、辺の構造と角の関係性を視覚的に把握することが推奨されます。特に、斜辺、直角、対辺を含む組み合わせ（斜辺，直角，対辺，角）の扱いや、SSA による不確定な状況を把握（アングラ、アムビギアス キース）することが重要です。

## 2. 合同条件の 3 段階構造

三角形が合同であるための条件は、以下の 3 つの段階に分けて理解できます。

### （1）辺が全て同長さの条件（SSS）
SSS は 3 つの辺が全て等しい場合です。これは、3 つの辺の長さが固定されていると、その三角形の形状は一意に決まります。つまり、3 つの辺の長さが等しい 2 つの三角形は、必ず合同です。

### （2）角と辺の組み合わせ（SAS、ASA、AAS）
SAS は 2 辺とその挟み角が等しい場合です。この条件では、挟み角で固定された 2 つの辺が、回転移動（ピボット）を繰り返しても、3 つ目の点（角の交点）は 1 つのみに固定されます。
ASA は 2 角とその挟み辺が等しい場合です。この条件では、2 つの角の大きさが固定されると、3 つ目の角の大きさは自動的に決定されます（三角形の内角の和は常に 180 度であるため）。残りの辺や角の構造もこれに伴って固定的になります。
AAS は 2 角と 1 つの対辺が等しい場合です。これは ASA と同等に形状が決まるため、合同の条件となります。ただし、AAS の場合、2 つの角が等しいことで残りの角（3 つ目の角）も必然的に等しくなります。

### （3）2 辺 1 角の挟み角・対辺の違い
SAS（2 辺 1 角で、角が挟む）と、SSA（2 辺 1 角で、角が挟まない）の違いに注意してください。SSA は形状が不定であり、したがって合同の条件にはなりません。この不確定な状況を把握することが重要です。特に、斜辺（最も長い辺）が固定され、角とその対辺が与えられた場合は、2 つの三角形が存在し得る（アムビギアス）場合がある。したがって、合同条件として扱うべきではありません。

## 3. 三角形の合同条件の証明方法（概略）

各合同条件がなぜ成立するのか、幾何学的な背景から簡単に説明します。

### SSS (Side-Side-Side)
3 つの辺が固定されている場合、三角形は一意に決まります。このことは、三角形の不等式（3 辺の不等式：最短辺×2 > 最大辺）を満たす限り、形状が確定するからです。
三角形の 3 辺の長さが全て等しいと仮定すると、その三角形の形状は一意に定まります。
したがって、合同であることは必然です。

### SAS (Side-Angle-Side)
2 つの辺とその挟み角が等しい場合、形状は一意に決まります。挟み角が、その 2 辺の端点にあるため、辺の長さが決まると、残りの頂点は固定されます。
つまり、残りの辺の長さは固定され、形状は一意に決まります。

### ASA (Angle-Side-Angle)
2 つの角とその挟み辺が等しい場合、三角形は一意に決まります。2 つの角の大きさが決まると、残りの角の大きさは自動的に決まります（内角の和）。このため、辺の構造も自動的に決まります。

### AAS (Angle-Angle-Side)
2 つの角と 1 つの対辺が等しい場合、これは ASA に変換可能です。2 つの角が等しい場合、残りの角も等しくなるため、三角形の内角の和（180 度）を用いて、3 つ目の角が求められます。これによって、三角形の構造が固定されます。

### SSA (Side-Side-Angle)
SSA は合同条件にはなりません。2 辺と 1 つの角が等しい場合、その角が挟んでいる（SAS）場合もあれば、挟んでいない（SSA）場合もあります。角が挟んでいない場合、三角形の形状は一意に決まらない場合があります。
特に、対辺の長さが、固定された角の対辺の長さよりも短い場合、2 つの三角形が存在することもあり得ます。この状態を「アムビギアス キース」と呼称し、不確定な形状になります。

## 4. 三角形の合同条件の例題

### 例題 1：SSS 合同条件の確認
以下の三角形について、三角形が合同であることを確認してください。
条件：三角形 $ABC$ と三角形 $DEF$ において、
- $\overline{AB} = \overline{DE} = 5$
- $\overline{BC} = \overline{EF} = 7$
- $\overline{AC} = \overline{DF} = 9$

解答：与えられた条件から、三角形 $ABC$ と三角形 $DEF$ は 3 辺がそれぞれ等しく、したがって SSS 条件を満たしています。
したがって、三角形 $ABC$ は三角形 $DEF$ と合同です。記号では $\triangle ABC \equiv \triangle DEF$ と表します。

### 例題 2：SAS 合同条件の確認
以下の三角形について、三角形が合同であることを確認してください。
条件：三角形 $ABC$ と三角形 $PQR$ において、
- $\overline{AB} = \overline{PQ} = 6$
- $\overline{BC} = \overline{QR} = 8$
- $\angle B = \angle Q = 45^\circ$

解答：この条件は、2 辺とその挟み角が等しいため SAS 条件を満たしています。
したがって、三角形 $ABC$ は三角形 $PQR$ と合同です。記号では $\triangle ABC \equiv \triangle PQR$ と表します。

### 例題 3：AAS 合同条件の確認
以下の三角形について、三角形が合同であることを確認してください。
条件：三角形 $XYZ$ と三角形 $UVW$ において、
- $\angle X = \angle U = 30^\circ$
- $\angle Y = \angle V = 60^\circ$
- $\overline{ZV} = \overline{YW} = 5$

解答：この条件は、2 角と対辺が等しいため AAS 条件を満たしています。
したがって、三角形 $XYZ$ は三角形 $UVW$ と合同です。記号では $\triangle XYZ \equiv \triangle UVW$ と表します。
残りの辺 $\overline{XY} = \overline{UV}$、$\overline{YZ} = \overline{VW}$、$\overline{ZX} = \overline{WU}$、$\angle Z = \angle W = 90^\circ$ を推定できます。

## 5. 三角形の合同条件の練習問題

### 問題 1：SSS 合同条件の適用
以下の条件を満たす 2 つの三角形について、合同であることを示してください。
- 辺の長さ：$AC = 4$、$BC = 5$、$AB = 6$
- 辺の長さ：$A'B' = 4$、$B'C' = 5$、$A'B' = 6$

解答：3 辺の長さがそれぞれ等しいため、SSS 条件を満たします。合同です。

### 問題 2：SAS 合同条件の適用
以下の条件を満たす 2 つの三角形について、合同であることを示してください。
- 辺の長さ：$AC = 3$、$AB = 5$
- 辺の長さ：$A'B' = 3$、$A'B' = 5$
- 角：$\angle A = \angle A' = 90^\circ$
解答：2 辺とその挟み角が等しいため、SAS 条件を満たします。合同です。
