## 概念説明

代数の基礎となる「文字と式」は、具体的な数を一般化するための記号体系であり、数学的表現を抽象化する重要な概念である。代数では「文字」が特定の数を表すために用いられ、「式」は数や文字の演算によって構成される構造として定義される。

**文字（変数名）**は、未知の数や任意の数を表すための記号である。x, y, z, a, b, c などが一般的に用いられる。文字の前に係数が付く場合（例：3x）、その係数はその文字が表す数の倍数または部分を示す（3x は「3 という数を x という数に掛けたもの」）。

**式（数式）**は、数字、文字、演算記号（+, -, ×, ÷, べき乗）を用いて構成された数学的表現である。式は計算を行うための指示や関係を表す。単項式は文字の倍数のみ、二項式は複数の項が足された構造、多項式は複数の項からなる式を指す。

**代入**とは、式に文字が含まれている場合に、その文字に具体的な数を代入して式の値を計算する操作である。代入の記法は → または = によって表現されることが多く、代数計算の基礎となる手続きである。

**代数表現の利点**は、未知の値や任意の値を用いて一般性を表せる点にある。具体例：「リンゴ 1 個の値段」を a とすると、「リンゴ 3 個の値段」は 3a で表せる。同様に「リンゴ 1 個の値が b、オレンジ 1 個の値が c」とすると、「リンゴ 3 個とオレンジ 2 個の総額」は 3b + 2c で表せる。

代数式における優先順位は、括弧内を先に計算、次にべき乗、次に乗算・除算、最後に加算・減算の順で行われる。この順序を守らないと誤った結果になるため、計算時の注意点が重要である。

## 例題

### 例題 1

次の文字式を言葉で説明し、具体的な数を入れて計算せよ。

**問題:**  a = 5, b = 2 のとき、次の式の値を求めよ。

a) 3a + b
b) 2(a + b)
c) a² - b²

**解法と解答:**

a) 3a + b における a = 5, b = 2 を代入する
    3 × 5 + 2 = 15 + 2 = 17
    答え：17

b) 2(a + b) における a = 5, b = 2 を代入する
    2 × (5 + 2) = 2 × 7 = 14
    答え：14

c) a² - b² における a = 5, b = 2 を代入する
    5² - 2² = 25 - 4 = 21
    答え：21

**重要な手続き:** 代入の時には、文字がある式の中で係数部分を注意深く読み取り、括弧がなければ乗算、括弧があれば括弧内の計算を先に実行する。指数（a²）は a を 2 回掛けたものとして計算する。

### 例題 2

次の文章から文字式を作製し、具体的な数でその値を求めよ。

**問題:** 「リンゴの個数を x、オレンジの個数を y とすると、リンゴ x 個とオレンジ y 個を買うための総額は、リンゴ 1 個の価格が 120 円、オレンジ 1 個の価格が 80 円である。」

a) 文字式で総額を表せ（円単位）。
b) x = 3, y = 5 のときの総額を計算せよ。
c) x = 0, y = 8 のときの総額を計算せよ。

**解法と解答:**

a) リンゴ 1 個 120 円、オレンジ 1 個 80 円
    総額 = 120 × x + 80 × y
    文字式：120x + 80y
    答え：120x + 80y

b) x = 3, y = 5 を代入する
    120 × 3 + 80 × 5 = 360 + 400 = 760
    答え：760 円

c) x = 0, y = 8 を代入する
    120 × 0 + 80 × 8 = 0 + 640 = 640
    答え：640 円

**重要な手続き:** 文章から変数を定義し、式を構築するプロセスは、実用的な数学問題に直面する際に重要である。代入時に 0 の値を適切に処理する（0 × 数 = 0）ことも注意が必要である。

## 練習問題

### 問題 1

**問題:** 文字式を完成させよ。以下の式を言葉で説明し、値を計算せよ。

a) a + 2b + 3c において a = 1, b = 2, c = 3 のとき
   計算結果を求めよ。

b) 3(x - y) において x = 8, y = 4 のとき
   括弧の計算結果を注意深く計算し、答えを求めよ。

c) a² + b² において a = 4, b = 5 のとき
   指数計算の結果を注意深く求めよ。

**解答:**

a) a + 2b + 3c における a = 1, b = 2, c = 3 を代入する
   1 + 2 × 2 + 3 × 3 = 1 + 4 + 9 = 14
   答え：14

b) 3(x - y) における x = 8, y = 4 を代入する
   3 × (8 - 4) = 3 × 4 = 12
   答え：12

c) a² + b² における a = 4, b = 5 を代入する
   4² + 5² = 16 + 25 = 41
   答え：41

**採点基準:**
- 正しい代入手順を示す
- 指数計算を正しく実行する（a² = a × a）
- 括弧内の計算を先に実行する
- 乗算の結果を正確に計算する
- 最終的な計算結果が正しい

### 問題 2

**問題:** 次の文章から式を作り、値を求めよ。

「車の距離を d メートル、速度を v メートル/秒とする。時間が t 秒後の距離は 100t + 500 メートルである。」

a) t = 3 のとき距離 d を計算せよ。
b) t = 5, d = 700 のとき、式 100t + 500 の値を求めよ。

**解答:**

a) t = 3 のとき、d = 100t = 100 × 3 = 300 メートル
   答え：300 メートル

b) 100t + 500 の値を t = 5 で代入する
   100 × 5 + 500 = 500 + 500 = 1000
   答え：1000 メートル

**採点基準:**
- t の値を正しく文字式に代入する
- 100 × t の計算を正確に実行する
- 500 の定数項を足す計算を忘れない
- 乗算の優先順位を正しく守る

### 問題 3

**問題:** 以下の文字式を言葉で説明し、値を求めよ。

a) 2(x + y) - z において x = 2, y = 3, z = 1
b) a - 2b + 3c において a = 7, b = 2, c = 4
c) (a + b)² において a = 1, b = 2

**解答:**

a) 2(x + y) - z の値
   2 × (2 + 3) - 1 = 2 × 5 - 1 = 10 - 1 = 9
   答え：9

b) a - 2b + 3c の値
   7 - 2 × 2 + 3 × 4 = 7 - 4 + 12 = 15
   答え：15

c) (a + b)² の値
   (1 + 2)² = 3² = 9
   答え：9

**採点基準:**
- 括弧内の計算を先に実行する
- 乗算を足算より先に実行する（乗算優先）
- 負数を扱う（2b は 2 × b）
- 指数計算を適切に実行する
- 計算ミスがない

### 問題 4

**問題:** 次の文字式を完成させよ。式の意味を説明し、値を計算せよ。

a) 2a - a + b において a = 5, b = 3
b) 3(a - b) + 2c において a = 7, b = 3, c = 2
c) (a × b) ÷ c において a = 6, b = 8, c = 4

**解答:**

a) 2a - a + b の値
   2 × 5 - 5 + 3 = 10 - 5 + 3 = 8
   答え：8

b) 3(a - b) + 2c の値
   3 × (7 - 3) + 2 × 2 = 3 × 4 + 4 = 12 + 4 = 16
   答え：16

c) (a × b) ÷ c の値
   (6 × 8) ÷ 4 = 48 ÷ 4 = 12
   答え：12

**採点基準:**
- 文字の係数を正しく処理する
- 括弧内の計算を先に実行する
- 乗算・除算の順序を正しく守る
- 最終的な計算結果が正しい

### 問題 5

**問題:** 次の文字式を言葉で説明し、値を求めよ。

a) x² - y² において x = 6, y = 3
b) 2x + 3y において x = 4, y = 2
c) (x + y)(x - y) において x = 5, y = 3

**解答:**

a) x² - y² の値
   6² - 3² = 36 - 9 = 27
   答え：27

b) 2x + 3y の値
   2 × 4 + 3 × 2 = 8 + 6 = 14
   答え：14

c) (x + y)(x - y) の値
   (5 + 3) × (5 - 3) = 8 × 2 = 16
   答え：16

**採点基準:**
- 指数計算を適切に実行する
- 乗算の優先順位を遵守する
- 括弧の計算順序を正しく処理する
- 文字式の構造を理解している

### 問題 6

**問題:** 次の式を完成させよ。値を計算せよ。

a) x + y - z において x = 10, y = 5, z = 7
b) 2(x² + y²) において x = 3, y = 4
c) ab ÷ c において a = 8, b = 6, c = 12

a) x + y - z の値
   10 + 5 - 7 = 15 - 7 = 8
   答え：8

   2 × (3² + 4²) = 2 × (9 + 16) = 2 × 25 = 50
   2 × (9 + 16) = 2 × 25 = 50
   答え：50

b) 2(x² + y²) の値

a) 文字式で総額を表せ（円単位）。
b) x = 3, y = 5 のときの総額を計算せよ。
c) x = 0, y = 6 のときの総額を計算せよ。

**重要な手続き：**文章から変数を定義し、式を構築するプロセスは、実用的な数学問題に直面する際に重要である。代入時に 0 の値を適切に処理する（0 × 数 = 0）ことも注意が必要である。

**解法と解答:**

a) リンゴ 1 個 120 円、オレンジ 1 个 80 円
    総額 = 120 × x + 80 × y
    文字式：120x + 80y

b) x = 3, y = 5 を代入する
    120 × 3 + 80 × 5 = 360 + 400 = 760 円

c) x = 0, y = 6 を代入する（注意：ユーザーが 6 に訂正した可能性あり）

- 文字の係数を正しく処理する
- 代入時に 0 の値を適切に処理する
