## 概念説明

**角度**とは、2 本の直線が一点（頂点）で交わってできる開口の大きさを表す量です。この大きさを「度（°）」という単位で表します。

**分度器（プロテクト）**は、角度の大きさをはかるための道具です。円形のスケールに、0〜180 度の目盛りが記載されています。使い方を覚えて、様々な角度の大きさを正確に読み取ることができます。

### 角度の種類と基準
| 角度の種類 | 大きさの条件 |
|----------|-------------|
| **直角** | ちょうど 90 度 |
| **鋭角** | 0 度より大きく、90 度より小さい（0° < 角度 < 90°） |
| **鈍角** | 90 度より大きく、180 度より小さい（90° < 角度 < 180°） |
| **平角** | ちょうど 180 度 |
| **一周角** | ちょうど 360 度 |

分度器を使って角度を測るとき、直角のとき 90 度、平角のとき 180 度、一周角のとき 360 度です。

**重要な注意点**: 分度器には「0〜180 度」と「0°/360°」の 2 つの目盛りがあります。どの目盛りを使い、どの数字を読み取るかは、分度器の中心（原点）を位置合わせする前に確認してください。

## 例題

### 例題 1：直角・鋭角・鈍角の判別
**（1）** 下図の角度を分度器を使って測定しましょう。

```
    A
   /|
  / |
B/  |
```
この角（∠ABC）が直角のとき、分度器の目盛りは「0 度」から「90 度」になります。

**解答**: ∠ABC = 90°。分度器の「0 度」の目盛りを AB に合わせ、BC の位置で目盛りを読み取る。このとき 90 を示す。

**（2）** 下図の角度は直角ですか？それとも鋭角ですか？

```
      A
     /|
    / |
   /  |
  /   |
 B    |
```
この角（∠ABC）の開口が 90°より小さい場合。

**解答**: ∠ABC は鋭角です。分度器で測定すると、85°や 45°など 90°未満の値が出ます。

**（3）** 下図の角度は直角ですか？それとも鈍角ですか？

```
         A
        /|
       / |
      /  |
     /   |
    /    |
   B-----C
```
この角（∠ACB）が直角より大きい場合。

**解答**: ∠ACB は鈍角です。分度器で測定すると、100°や 150°など 90°より 180°未満の値が出ます。

### 例題 2：分度器を使って角度を測る

**（1）** 分度器を使って、∠PQRの大きさを測ります。
```
       P
      /|
     / |
    /  |
   /   |
Q  /    |
    \   |
     \  |
      \ |
       \|
        R
```
手順：
1. 分度器の中心（丸い穴）を Q の点に合わせる
2. 分度器の 0 度目盛りを QP の辺に合わせる
3. QR の辺が示す数字を読み取る
4. どちらの目盛りを使うか決める

**実測値**: 65 度（分度器の 0 度目盛りから 65 まで数える）

**解答**: ∠PQR = 65°。どちらの目盛りを使っても、同じ 65度の値を示します（分度器の中心を合わせて、0 度の位置を PQ に合わせることで、QR と 0 度目盛りが一致するか、QR の位置が 65 度を示すかを確認します）。

**（2）** 0 度と 360 度の位置について確認します。
```
        360度
       /
      /0度
     /
    /
   /
```
0 度と 360 度は同じ位置を示します。分度器を一周回ったら 360 度になります。

**解答**: 0 度と 360 度は同じ位置を表します。「0 度」と表記して、360 度まで読み取ります。

**（3）** 分度器の 0 度目盛りの位置を確認します。
分度器には「0 度・180 度」と「0°/360°」の 2 つの目盛りがあります。どちらを使うか確認する：

```
  0°/360°        180°
    │            │
    └──┬───┬────────┘
       │   │
```
実際の角度を測る場合、分度器の中心を頂点に合わせ、0 度の位置を 1 辺に合わせ、もう 1 辺の位置を読み取ります。0 度・180 度目盛りを使うか、0°/360 度目盛りを使うかは、分度器の設計により異なりますが、多くの分度器では、0 度・180 度と 180 度・0 度の 2 つの目盛りがあります。どちらを使うかは、分度器の設計により異なります。

**実例**: 内側に「0 度・180 度」と書かれている分度器なら、0 度から数えます。外側に「0 度・180 度」と書かれている分度器なら、0 度から数えます。

### 問題 1：角度の種類を分類しよう

以下の角度が「直角」「鋭角」「鈍角」「平角」「一周角」のどれに当たるかを答えなさい。

（1）90°
（2）60°
（3）105°
（4）180°
（5）360°
（6）45°
（7）175°
（8）0°

**解答**:
（1）直角（90 度）
（2）鋭角（0° < 60° < 90°）
（3）鈍角（90° < 105° < 180°）
（4）平角（180 度）
（5）一周角（360 度）
（6）鋭角（0° < 45° < 90°）
（7）鈍角（90° < 175° < 180°）
（8）0°（0度と 360 度は同じ位置）

**採点基準**:
- 各度数は、直角（90°）、鋭角（0° < 角度 < 90°）、鈍角（90° < 角度 < 180°）、平角（180°）、一周角（360°）の条件に一致しているか
- 0°は「0 度」として正解
- 分度器の 0 度目盛りの位置と、360 度目盛りの位置は、同じ位置を示します

### 問題 3：歴史的背景を学ぶ

**背景**: 分度器は、古代の天体観測や地図作成などで使用されました。1802 年、イギリスの天文学者ジョン・ハドレイが、天体観測のための角度測定装置「クロービッド」として分度器を考案しました。これは、天体の位置や軌道の角度を正確に測定するためのものでした。

**（1）** 古代バビロニアやエジプトでは、角度を測るために「分度器」のような道具は使われていましたか？

**解答**: いいえ、古代バビロニアやエジプトでは、角度を測るために「分度器」のような道具はありませんでした。彼らは主に直角（90°）や平角（180°）を測るために、簡易的な道具や幾何学的な方法を用了。

**（2）** 分度器という名称は、何語で「分度」から来ていますか？

**解答**: 分度器という名称は、古代バビロニアやエジプトで使われた「度（degree）」という言葉に由来します。

## まとめ

角度は、直角（90 度）、鋭角（0°〜90°）、鈍角（90°〜180°）、平角（180 度）、一周角（360 度）に分類されます。

分度器は、2 本の直線が交わってできる角度の大きさを測る道具です。中心を頂点に合わせ、0 度目盛りの位置を 1 辺に合わせ、もう 1 辺の位置を読み取ることで、角度の大きさを正確に測定できます。

歴史的には、古代バビロニアやエジプトでは、分度器ではなく、簡易的な道具や几何学的な方法で角度を測るために使われていました。1802 年、イギリスの天文学者ジョン・ハドレイが、角度測定装置「クロービッド」を考案しました。
