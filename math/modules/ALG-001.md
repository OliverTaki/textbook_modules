## 概念説明

多項式の展開と因数分解は、代数における最も基本的で重要な演算技術です。本モジュールでは、以下の点を理解し、確かな技能を身につけます。

### 分配法則（分配則）
多項式が括弧を含む場合、それを開く（展開する）ためには、分配法則 $$a(b+c) = ab + ac$$ が基本となります。括弧内の各項をそれぞれ掛ける操作です。

例：$$2(x+3) = 2x + 6$$

$$3(2x-5) = 6x - 15$$

### 多項式同士の掛け算
2 つ以上の多項式を掛け合わせる場合、各係数をそれぞれの項と掛け合わせます。

例：$$(x+2)(x+3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$$

### 乗法公式
以下の公式を覚えておくと、展開が素早く行えます。

| 公式 | 展開式 |
|------|--------|
| 差の 2 乗 | $$(a-b)^2 = a^2 - 2ab + b^2$$ |
| 和の 2 乗 | $$(a+b)^2 = a^2 + 2ab + b^2$$ |
| 和と差の積 | $$(a+b)(a-b) = a^2 - b^2$$ |

### 因数分解とは
ある多項式を、他の多項式の積の形に書き換える作業です。展開と逆の操作です。

例：$$x^2 + 5x + 6 = (x+2)(x+3)$$

因数分解を正しく行うためには、展開の知識を確実に理解しておく必要があります。

### 因数分解の手法
多項式の構造に応じて適切な手法を選定します。

- **共通因数の引き出し**：すべての項に共通する因数がある場合
- **2 項の場合**：差の 2 乗、和・差の積、共通因数の引き出し
- **3 項の場合**：2 乗の式（$$ax^2 + bx + c$$）は因数分解の公式や、因数定理、グラフの交点、共通因数引き出し + 分组の組み合わせ

### 因数定理
多項式 $$P(x)$$ において、$$P(a) = 0$$ ならば、$$x-a$$ は $$P(x)$$ の因数である。

つまり、$$P(0) = 0$$ のとき、$$x$$ は因数となります。

### 平方完成
一般形二次式 $$ax^2 + bx + c$$ を、$$(x-h)^2 + k$$ の形に書き直すことを平方完成と言います。

これは二次関数の解析や最適化問題で重要になります。
$$ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 + \left(c - \frac{b^2}{4a}\right)$$

### グループ分解
4 項多項式の因数分解では、2 項ずつに分けて共通因数を探す手法が有効です。

例：$$ax + ay + bx + by = a(x+y) + b(x+y) = (a+b)(x+y)$$

### まとめ
- 展開・因数分解は逆な作業
- 分配法則が基礎
- 乗法公式を覚える
- 式の形に合わせて適切な手法を選んで解く
- 展開の訓練は因数分解の力向上に直結

## 例題

### 例題 1：基本的な展開

**問題：** 次の式を展開せよ。

$$(x-3)(x+4)$$

**解答過程：**

1. 分配法則に従って展開する。
2. 第一括弧の $$x$$ を第二括弧の各項に掛ける：$$x(x+4) = x^2 + 4x$$
3. 第二括弧の $$-3$$ を第一括弧の各項に掛ける：$$-3(x+4) = -3x - 12$$
4. 2 つの結果を足し合わせる：$$(x^2 + 4x) + (-3x - 12) = x^2 + 4x - 3x - 12$$
5. 同类項をまとめて整理：$$x^2 + x - 12$$

**最終答え：** $$x^2 + x - 12$$

**解説：** ここでは分配法則を丁寧に 2 回使いました。分配法則を使うのが基本であるため、公式を思い出し、式を直接展開する練習が有効です。

---

### 例題 2：差の 2 乗の公式の利用

**問題：** 次の式を展開せよ。

$$(2x-5)^2$$

**解答過程：**

1. 差の 2 乗公式 $$(a-b)^2 = a^2 - 2ab + b^2$$ を適用します。
2. $$a = 2x, b = 5$$ として代入：
3. 展開：$$(2x)^2 - 2(2x)(5) + 5^2 = 4x^2 - 20x + 25$$
4. 計算：$$20x$$ は、$$2 \cdot 2x \cdot 5 = 20x$$ で正しい。

**最終答え：** $$4x^2 - 20x + 25$$

**注意点：** 多くの人が $$20x$$ を $$2 \cdot 5x = 10x$$ と計算してしまい、これがよくあるミスです。公式の $$-2ab$$ 部分は、$$\color{red}{2ab}$$ のように **2 倍** を忘れないことです。

---

### 例題 3：要因分解

**問題：** 次の式を因数分解せよ。

$$x^2 + 6x + 8$$

**解答過程：**

1. $$x^2$$ の係数が 1 の場合、因数の定数は $$x^2$$ の係数 1 になります。
2. 2 つの因数の積が定数項 8、和が一次項係数 6 です。
3. 8 の因数の組み合わせ：
   - $$1 \times 8 = 8, 1 + 8 = 9$$（×）
   - $$2 \times 4 = 8, 2 + 4 = 6$$（◎）
4. 因数を掛ける場合分け、$$2 + 4 = 6$$ となる $$2, 4$$ を選定。
5. 答え：$$(x+2)(x+4)$$

**解答：** $$(x+2)(x+4)$$

**確認：** 展開して検算：$$x^2 + 4x + 2x + 8 = x^2 + 6x + 8$$（正し）

**解説：** 一次元の二次式の因数分解は、2 つの因数を探す問題です。定数項の因数の組み合わせを試し、和が一次項の係数となる組み合わせを選定します。

---

### 例題 4：共通因数を引く因数分解

**問題：** を因数分解せよ。

$$2x^2 - 8x$$

**解答過程：**

1. 2 つの項に共通する因数を探します。
2. $$2x^2$$ と $$-8x$$ の共通因数は $$2x$$ です。
3. 共通因数を引き出します：$$2x(x - 4)$$

**最終答え：** $$2x(x-4)$$

**解説：** 共通因数の引き出しは、因数分解の第一歩です。この後に式が単純化できればよいでしょう。$$2x(x-4)$$ は完全に因数分解されています。

---

### 例題 5：差の 2 乗公式の因数分解への応用

**問題：** 次の式を因数分解せよ。

$$x^2 - 16$$

**解答過程：**

1. 2 項で、どちらも平方形であることが分かる：$$(x)^2 - (4)^2$$
2. 差の 2 乗公式：$$(a^2 - b^2) = (a-b)(a+b)$$
3. $$a = x, b = 4$$ の場合、答え：$$(x-4)(x+4)$$

**最終答え：** $$(x-4)(x+4)$$

**解説：** 4 は $$4^2 = 16$$ となるため、$$x^2 - 16$$ は $$x^2 - 4^2$$ と見なせます。平方を因数分解する公式を使う場合は、平方になっていることを確認してください。
