# 多項式の展開と因数分解

## 導入

多項式（多项式）は、数学（Mathematics）の基礎となる概念であり、方程式（Equation）を解いたり、関数を処理したりするために不可欠です。
展開（Expanding）とは、括弧を外して項を足し合わせることを指します。例えば、(x + 1)(x - 1) は x² - 1 になりますが、これは分配法則（Distributive Law）によって展開されるものです。

因数分解（Factorization）とは、多項式を、その要素に分解する逆の操作です。例えば、二次方程式 ax² + bx + c = 0 を解く際、因数分解が重要です。
このモジュールでは、展開の手法、因数分解の手法、計算上の注意点、および文化的な違いについて学びます。

## 概念説明

展開と因数分解は、多項式において最も重要な操作です。数学の用語を正確に理解するために、以下の法則を理解します。

1.  **分配法則 (Distributive Law):**
    a(b + c) = ab + ac
    係数を括弧内の各項に掛けます。
    
2.  **完全平方の展開:**
    (a + b)² = a² + 2ab + b²
    (a - b)² = a² - 2ab + b²
    2ab は共通項です。正負に注意します。
    
3.  **平方差の公式:**
    (a + b)(a - b) = a² - b²
    積が差になる場合に使います。
    
4.  **因数分解の基本:**
    a(x + y) = ax + ay 型
    公約数（Common Factor）を探す。
    
5.  **二次式の因数分解:**
    x² + px + q 型。
    x² + (a + b)x + ab 型。
    因数 x + a と x + b の積になる。

## 例題

### 例 1: 展開

**問題:** (2x + 3)(x - 4) を展開して簡易化してください。

**解決策:**
分配法則を用いて、左側の括弧内の各項を右側の括弧に乗けます。
(2x + 3)(x - 4) 
= 2x(x - 4) + 3(x - 4) （分配の法則）
= 2x² - 8x + 3x - 12
= 2x² - 5x - 12

**注意:**
項を組み合わせて -8x + 3x を -5x にする必要があります。

### 例 2: 因数分解

**問題:** x² - 5x + 6 を因数分解してください。

**解決策:**
二次式 ax² + bx + c を x² + (a + b)x + ab の形に変換します。
ここでは x²の係数が 1 です。
2 項の積が一定である場合、a と b として積が 6 になり、相加法則（和）が -5 になるペアを探します。
6 の因数対は (1, 6), (2, 3) です。
1 + 6 = 7 (違う、マイナスが必要) -> (-1, -6) -> 7
2 + 3 = 5 -> (-2, -3) -> -5 に一致。
よって、(x - 2)(x - 3) です。

## 演習

以下の問題を解いてください。答えを単純化し、不要な括弧は外してください。計算ミスがないか確認してください。

1.  **問題:** (x + 2)(x + 3)(x - 4) の積の項を計算してください。
    *   *ヒント:* (x + 2)(x + 3) 部分を先に展開、その後 (x - 4) で展開します。
    *   *答え:* x³ - 2x² - 5x - 24
2.  **問題:** x² - 9 を因数分解してください。
    *   *ヒント:* 平方差の公式 (a² - b²) = (a - b)(a + b) を使いましょう。
    *   *答え:* (x + 3)(x - 3)
3.  **問題:** 2x² + 4x - 6 を因数分解してください。
    *   *ヒント:* まず、2 を外します（公約数抽出）。その後、x に関する式を因数分解します。
    *   *答え:* 2(x + 3)(x - 1)

## 文化差

数学の教育方法や記符号（Notation）は、世界によって異なります。

1.  **展開の順序:**
    日本では、分配法則（配列）を「配列法」として厳密に教えることが多いです。
    特に、括弧を外す順序（先に乗じる項、後に乗じる項）を教えることがあります。
    一方、西洋では代数表現として「展開順序」は「右から左に並べる」という一般的慣習に従うことが多いです。
    
2.  **変数の使用:**
    日本では「x, y」をそのまま使用しますが、特定の文化圏では「X₁, X₂」というように添字を使うこともあります。
    また、係数の表記（分数）で、日本では「½」、西洋では「1/2」が一般的です。
    
3.  **因数分解の解法:**
    日本では因数分解の公式（型）に当てはめることを重視し、式を変形して解く手順（手順）を踏む練習が多いです。
    西洋では因数分解の背後にある数学的構造や一般定理を重視し、証明の練習が多いです。

## 注意

計算上、必ず注意するべき点があります。

1.  **括弧（Parentheses の役割）:**
    括弧は「優先順位」を定義します。展開しないままの項は、括弧の中に残します。必ず分配した結果を確認してください。
    
2.  **記号の意味:**
    「・」は積を表しますが、省略する場合もあります。
    「+」は足し算、「-」は引き算です。
    計算時に符号を間違えると、答えが逆になります。
    特に「-5x + 2x」は -3x になります。
    「2x + (-3x) = -x」とも書くことができます。
    
3.  **多項式の次数:**
    次数が高い式ほど、計算ミスが発生しやすいです。
    項を増やしながら計算を続けないでください。
    必ず 1 つずつ、括弧内の項を分配していきましょう。
    
    計算の合否は、解答の簡易性が基準となることが多いです。
    余計な項が含まれている場合、評価が下がります。

## 歴史

多項式の展開の歴史は、数学の発展と深く関連しています。

1.  **古代の起源:**
    古代バビロニアやエジプトに、多項式の概念に近いものがあったと考えられています。
    しかし、代数として体系化されたのは、イスラム圏の学者（アル＝フワリズミー）が「アル＝ジャブラ」を記述した 9 世紀頃です。
   那时的数学は、幾何学的な問題解決に使用されていました。<br>
    <br>
2.  **ルネサンス:**
    16 世紀イタリアの数学家（フィエボ、ピアノ）が、代数的解法を発展させました。
    この時期、代数的な式を展開する手法は、数学の基礎として確立され始めました。<br>
    <br>
    ルネサンス期、多項式の展開は芸術と同じく、理論化されました。<br>
    <br>
3.  **近代:**
    近代数学は、展開のアルゴリズムや因数分解の定理が整い、計算機の発展に寄与しました。<br>
