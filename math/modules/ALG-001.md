## 概念説明

多項式とは、係数に定数を含み、変数の積の総和として表される式のことです。多項式の展開とは、括弧内の和・差に掛ける分配法則を適用して、括弧をはずし、項同士の和に書き換える操作です。例えば、(a + b)(c + d) における展開では、まず a をかっこ内の各項に掛けて ac + ad, 次に b をかっこ内の各項に掛けて bc + bd となり、ac + ad + bc + bd という形になります。

因数分解とは、元の式をいくつかの単一多項式の積に書き換える逆操作です。展開と因数分解は同じ式変形を正逆に行う対概念です。展開法則の「分配法則 a(b + c) = ab + ac」を用いる逆方向に思考し、ab + ac = a(b + c) のように書き換える操作です。

一般的な展開パターンには以下があります：
- 二項同乗法の二乗：(a + b)² = a² + 2ab + b²
- 乗法の分配法則：k(m + n) = km + kn
- 多項式同士：(a + b + c)(d + e) = ad + ae + bd + be + cd + ce

一般的な因数分解パターンには以下があります：
- 共通因数因子の取り出し：ab + ac + ad = a(b + c + d)
- 平方差公式：a² - b² = (a - b)(a + b)
- 完全平方式：a² ± 2ab + b² = (a ± b)²
- 因数分解のグループ化：ab + ac + bd + bc = a(b + c) + b(c + d) = ...

## 例題

### 例題 1

**問題**: (x + 3)(x - 2) を展開して整有理式に整理せよ。

**解答**:

分配法則に従って展開します：
\[
\begin{aligned}
(x + 3)(x - 2) &= x \cdot x + x \cdot (-2) + 3 \cdot x + 3 \cdot (-2) \\
&= x^2 - 2x + 3x - 6 \\
&= x^2 + x - 6
\end{aligned}
\]

同類項をまとめることで x² + x - 6 という形になります。

**採点基準**:
- 分配法則の適用過程が明瞭 (4 ポイント)
- 同類項の整理が正解 (-6 ポイント)
- 最終結果 x² + x - 6 の形 (-6 ポイント)

### 例題 2

**問題**: -2x² + 8x - 6 の因数分解をせよ。

**解答**:

まず、各項の共通因数があるか確認します。各項の係数は -2, 8, -6 で、すべて偶数です。まず -2 で全体を約します：
\[
-2x^2 + 8x - 6 = -2(x^2 - 4x + 3)
\]

次に、x² - 4x + 3 を因数分解します。この多項式の根を求めます。因数分解の形式は (x - r₁)(x - r₂) で、r₁r₂ = 3, (r₁ + r₂) = 4 となるように求める必要がある。

3 の因数の組みは (1, 3), (-1, -3), (1, -3), (-1, 3) ですが、和が 4 になる組み合わせは 1 と 3 のみです。したがって：
\[
x^2 - 4x + 3 = (x - 1)(x - 3)
\]

よって：
\[
-2x^2 + 8x - 6 = -2(x - 1)(x - 3)
\]

**採点基準**:
- 共通因数 -2 の取り出しが正解 (-4 ポイント)
- 二次式 x² - 4x + 3 の因数分解が正 (-8 ポイント)
- 最終結果 -2(x - 1)(x - 3) が (-4 ポイント)

## 練習問題

### 問題 1

**問題**: (3x - 4)(2x + 1) を展開せよ。

**解答**: 分配法則を用います：
\[
(3x - 4)(2x + 1) = 3x \cdot 2x + 3x \cdot 1 + (-4) \cdot 2x + (-4) \cdot 1 = 6x^2 + 3x - 8x - 4 = 6x^2 - 5x - 4
\]

**採点基準**:
- 分配法則で 6x² + 3x - 8x - 4 として中間式を明記
- 同類項を -5x としてまとめる
- 最終結果 6x² - 5x - 4

### 問題 2

**問題**: x² - 10x + 21 を因数分解せよ。

**解答**:
21 の因数の組み合わせ：(1, 21), (3, 7), (21, 1), の組み合わせを考えながら -10 との一致を見る。

3 + 7 = 10 で和が正の場合、積が正、和が負の場合は負の符号を両方に。10 とは逆符号が -10 になる。したがって：
\[
x^2 - 10x + 21 = (x - 3)(x - 7)
\]

**採点基準**:
- 二次項係数が 1 の場合は、定数項を 2 個の整数積で分解
- その和が一次項の係数と一致する（符号を考慮して）
- 最終結果 (x - 3)(x - 7) で明記

### 問題 3

**問題**: a³ - 8 を因数分解せよ。

**解答**: これは「立方の差」として a³ - 2³ の形です。立方の差の公式 a³ - b³ = (a - b)(a² + ab + b²) を使用します：
\[
a^3 - 8 = a^3 - 2^3 = (a - 2)(a^2 + 2a + 4)
\]

**採点基準**:
- 立方の差の公式を明記
- (a - 2)(a² + 2a + 4) の形で最終結果を整理

## 問題生成ルール

**展開問題の生成ルール**:
1. 括弧内の項数を 2 項以内にする（初級）、多項式同乗の項数増は中級
2. 係数を整数に指定する（±1, ±2, ±3...）
3. 展開後の同類項を持つ問題に同類項の整理が必要
4. 一次係数は負になるか正になるかの両方を含む問題
5. 2 次式展開の場合は、二次項係数と定数項の値を調整

**因数分解問題の生成ルール**:
1. 共通因数がある問題：係数が偶数または 3 の倍数の組み合わせ
2. 二次式因数分解：定数項の因数の組み合わせが和/差になるように設計
3. 三次式の差：立方の差/和の公式を使う問題
4. 因数分解後の因数が単項式・多項式を混合
5. 全体に共通因数を有さない問題を設計

**採点基準の設計ルール**:
- 展開問題：分配法則の過程、同類項の整理、最終形
- 因数分解問題：共通因数の見出し、公式適用、最終因数の整理
- 計算ミス：符号の誤差、係数の計算、定数項の計算ミス

## 各国・文化的差異

**日本**: 
- 学習指導要領では多項式の展開・因数分解は高等学校数学 I の基礎として扱われる
- 平方差公式、完全平方式は必ず暗記する要求（例：a² - b², a² + 2ab + b²）
- 高校入試で頻出：展開 3 点、因数分解 7 点といった問題

**米国 (Common Core)**:
- Algebra I における多項式の操作
- 分配法則は「distributive property」として明確に命名
- 平方差公式は公式暗記より、因数分解のパターンとして理解
- 米国では因数分解後の共通因数取り出しも重要な評価ポイント

**共通性**:
- 代数学の基礎として国際的に共通
