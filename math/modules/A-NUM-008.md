## 分数（Fractions）の基礎知識と計算方法

### 🧮 1. 分数とは？（概念の理解）

**分数とは何か？**
分数は、全体を複数の等しい部分に分けたうちの、特定のいくつかを表す数です。

*   **分子（Numerator）(a)：** 分子に書かれている数。全体をいくつ分したか（分けた数）の総数、または取り出した部分の個数を表します。
*   **分母（Denominator）(b)：** 分母に書かれている数。全体をいくつに分けたか（分ける単位の総数）を表します。

**【例】**
ピザを8等分し、そのうちの3切れを食べた場合、これを「$\frac{3}{8}$」と表します。
*   分子 $3$：取った切れの数（3）
*   分母 $8$：ピザの全体の分割数（8）

**【重要】**
分母は $0$ にすることはできません。分母が $0$ の分数は数学的に定義できません。

### 🔄 2. 分数の比較と等しい分数（簡約化）

#### 🅰. 分数の大小比較
分母を揃えるか、分子分母に同じ数を掛けて分母を揃えるのが基本です。

*   $\frac{1}{2}$ と $\frac{2}{4}$ を比べたい場合：分母を $4$ に揃えます。
    $$\frac{1}{2} \times \frac{2}{2} = \frac{2}{4}$$
    $\frac{2}{4} = \frac{2}{4}$ なので、大小は等しいです。

#### 🅱. 既約分数（最も簡単な形）
分母と分子を**共通の最大公約数**で割って分母・分子を最小にすることです。これを**約分**といいます。

*   例：$\frac{12}{18}$
    *   分子 $12$ と分母 $18$ の最大公約数は $6$ です。
    *   $$\frac{12 \div 6}{18 \div 6} = \frac{2}{3}$$
    *   $\frac{2}{3}$ がこの分数の最も簡単な形（既約分数）です。

### ➕ 3. 分数の四則演算のルール

分数の計算は、「**分母・分子をそれぞれ計算する**」というルールを理解することが最も重要です。

#### 🅰. 【足し算と引き算】 $\rightarrow$ 【分母を揃える】
分母が異なる場合は、先に**通分（分母を通分すること）**を行い、分母を同じ値にしなければ計算できません。

$$\text{例: } \frac{1}{3} + \frac{1}{4}$$
1.  **通分化**: 分母 $3$ と $4$ の最小公倍数 $12$ を見つけます。
2.  **分子・分母を変換**:
    $$\frac{1}{3} = \frac{1 \times 4}{3 \times 4} = \frac{4}{12}$$
    $$\frac{1}{4} = \frac{1 \times 3}{4 \times 3} = \frac{3}{12}$$
3.  **計算**:
    $$\frac{4}{12} + \frac{3}{12} = \frac{4+3}{12} = \frac{7}{12}$$

#### 🅱. 【掛け算】 $\rightarrow$ 【分子同士、分母同士】
掛け算は最も簡単です。分子同士を掛け、分母同士を掛けます。

$$\text{例: } \frac{2}{5} \times \frac{3}{4}$$
1.  **計算**:
    $$\frac{2 \times 3}{5 \times 4} = \frac{6}{20}$$
2.  **約分**:
    $$\frac{6 \div 2}{20 \div 2} = \frac{3}{10}$$

#### 🅲. 【割り算】 $\rightarrow$ 【逆数への掛け算】
割り算は、「**割られる数**」に「**割る数の逆数**」をかけて計算します。

$$\text{例: } \frac{3}{4} \div \frac{6}{5}$$
1.  **逆数を取る**: $\frac{6}{5}$ の逆数は $\frac{5}{6}$ です。
2.  **掛ける**:
    $$\frac{3}{4} \times \frac{5}{6} = \frac{3 \times 5}{4 \times 6} = \frac{15}{24}$$
3.  **約分**:
    $$\frac{15 \div 3}{24 \div 3} = \frac{5}{8}$$

### 💡まとめ（フローチャート）

| 演算 | 記号 | 実行する操作 | キーワード |
| :--- | :--- | :--- | :--- |
| **足し算** | $+$ | 通分 $\rightarrow$ 分子を足す | **共通の分母**を見つける |
| **引き算** | $-$ | 通分 $\rightarrow$ 分子を引き算 | **共通の分母**を見つける |
| **掛け算** | $\times$ | 分子 $\times$ 分子 / 分母 $\times$ 分母 | **そのまま掛け算** |
| **割り算** | $\div$ | 割る数の**逆数**をかけて計算 | **逆数**を利用する |
