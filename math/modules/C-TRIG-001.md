# 三角関数の定義

## 概念説明

三角関数（正弦、余弦、正接）は、直角三角形の形状と辺の長さの比率から定義されるものです。これは三角形を相似させるための基礎的な性質から導出されます。

まず、鋭角 $\theta$ を持つ直角三角形 $\triangle ABC$ を考えます（頂点 $C$ が $90^\circ$）。このとき、$\theta$ の**対辺**の長さを $a$、**隣辺**の長さを $b$、**斜辺**の長さを $c$ とします。

1.  **正弦 ($\sin \theta$)**
    斜辺に対する比率として定義されます。
    \[
    \sin \theta = \frac{\text{対辺}}{\text{斜辺}} = \frac{a}{c}
    \]
    三角形が小さくても大きくても、角度 $\theta$ そのものが決まれば比率 $a/c$ は一定になる性質（相似の性質）を持っています。

2.  **余弦 ($\cos \theta$)**
    角度に隣接する辺に対する比率として定義されます。
    \[
    \cos \theta = \frac{\text{隣辺}}{\text{斜辺}} = \frac{b}{c}
    \]

3.  **正接 ($\tan \theta$)**
    斜辺を含まない、隣辺に対する比率として定義されます。
    \[
    \tan \theta = \frac{\text{対辺}}{\text{隣辺}} = \frac{a}{b}
    \]

重要なのは、この定義が「三角形のサイズ」に依存しないことです。すべての辺を 2 倍にしても、辺の比率は元の値のままであり、したがって三角関数値は変化しません。ただし、斜辺は分母に現れるため、定義自体は「比」の概念を厳密に扱う必要があります。これにより、角度の大きささえあればその比率は一意に定められることになります。

## 例題

### 例題 1
**条件:** 直角三角形 $\triangle ABC$ において、斜辺 $c$ の長さが 2 で、鋭角 $\theta$ が $60^\circ$（または $\pi/3$ ラジアン）です。
**手順:** 対辺 $a$ と隣辺 $b$ の長さを用いて三角関数の値を求めなさい。

**解:**
角度 $\theta = 60^\circ$ の直角三角形は、角度が $30^\circ, 60^\circ, 90^\circ$ に分割される特殊な直角三角形（$30-60-90$ 三角形）となります。この三角形は、**二等辺直角三角形（イソセラス）ではありません**（$30-60-90$ 三角形は不等辺直角三角形です）。

30-60-90 三角形の辺の比率を考えると、斜辺（90 の向かい）が $2$、$30^\circ$ に対しての直角が $1$、$60^\circ$ に対しての直角が $\sqrt{3}$ となります。
したがって、斜辺が 2 の場合、
- 対辺（30-60-90 の $\sqrt{3}$ の向き）は $\sqrt{3}$
- 隣辺（30-60-90 の $1$ の向き）は $1$

となり、三角関数値は以下のようになります。
- $\sin 60^\circ = \frac{\sqrt{3}}{2}$
- $\cos 60^\circ = \frac{1}{2}$
- $\tan 60^\circ = \frac{\sqrt{3}}{1} = \sqrt{3}$

### 例題 2
**条件:** 直角三角形 $\triangle XYZ$ において、$\theta = 45^\circ$（または $\pi/4$ ラジアン）で、斜辺が 2$\sqrt{2}$ です。
**手順:** 対辺と隣辺の長さを求め、定義を確認しなさい。

**解:**
角度が $45^\circ$ の直角三角形は「二等辺直角三角形」です。これは、角度が $45-45-90$ であり、対辺と隣辺の長さが等しいことを意味します。
ピタゴラスの定理を用いて辺の長さを求めます。対辺 $a$ と隣辺 $b$ が等しいと仮定すると $a=b$ であり、斜辺は $a\sqrt{2}$ です。
\[
\sqrt{a^2 + a^2} = 2\sqrt{2} \implies \sqrt{2a^2} = 2\sqrt{2} \implies a\sqrt{2} = 2\sqrt{2} \implies a=2
\]
よって、対辺 $a=2$ となり、三角関数値は以下のようになります。
- $\sin 45^\circ = \frac{2}{2\sqrt{2}} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$
- $\cos 45^\circ = \frac{2}{2\sqrt{2}} = \frac{\sqrt{2}}{2}$
- $\tan 45^\circ = \frac{2}{2} = 1$

## 演習問題

以下の問題に対し、解答を作成しなさい。

### 問題 1
直角三角形 $PQR$ において、$\angle P = 90^\circ$、$\angle Q = 30^\circ$、斜辺 $RQ$ の長さが 8 です。次の問いに答えよ。
(1) $PR$ と $PQ$ の長さを求め、$\sin 30^\circ$ および $\cos 30^\circ$ を計算せよ。

**解答ガイド:**
- 30-60-90 の性質を正しく適用しなさい（30 度の対辺は斜辺の半分）。
- 計算結果は $\sin 30^\circ = 1/2$, $\cos 30^\circ = \sqrt{3}/2$ を示そう。
- **採点基準**: 0 点（誤答）、3 点（計算ミス）、5 点（完全解答）、6 点（詳細解説あり）。

### 問題 2
角 $\theta$ が $90^\circ$ より大きい場合（鈍角）、および $\angle P = 90^\circ$ の三角形において $\theta$ が角度の定義範囲外の場合、三角関数の定義は成立するか？
(1) 鋭角の場合と比較し、定義が適用できない理由を記述せよ。

**解答ガイド:**
- 定義は直角三角形の「鋭角」に対して定義されるため、$\theta \ge 90^\circ$ の場合は基本的な辺の比では定義できない（座標平面での拡張は後）。
- 定義成立範囲（$0 < \theta < \pi/2$）について言及。

### 問題 3
選択肢問題: 次のうち、$\sin \theta$ の定義として正しいものはどれか？
(A) $\frac{\text{斜辺}}{\text{隣辺}}$
(B) $\frac{\text{対辺}}{\text{斜辺}}$
(C) $\frac{\text{斜辺 - 対辺}}{\text{隣辺}}$
(D) $\frac{\text{対辺}}{\text{隣辺} + \text{斜辺}}$

**解答ガイド:**
- 正しい定義を識別し、他の選択肢がなぜ間違っているか（例：隣辺/斜辺は余弦）を説明。
- 解答: 正。
- **採点基準**: (A) 1 点、(B) 5 点。

## 問題生成ルール (Generative Rules)

この教材から別の問題を作成する際のガイドラインです。

1.  **角度の調整**:
    - 鋭角 $0^\circ < \theta < 90^\circ$ の範囲内で、標準的な角度（$30^\circ, 45^\circ, 60^\circ$）を変えて生成しなさい。これらの角度は特殊な直角三角形の性質に基づき、正解が固定されます。
    - 計算機を使用せずに解けるよう、角度の選択に注意を払うこと。

2.  **辺の長さのスケーリング**:
    - 斜辺の長さを $L$ とすると、$0 < L$ である任意の長さを選べばよいが、簡潔さを考慮し、整数や簡易的な根号（$\sqrt{2}, \sqrt{3}$ など）を含む長さを選ぶこと。
    - 「斜辺 10」など、具体的な値を用いて学生の計算を促す問題も作成可能だが、比率の概念を理解させるためには抽象的な値も使用してよい。

3.  **三角形の種別**:
    - 「30-60-90 三角形」および「45-45-90 三角形」の概念を混同させないよう、問題文や解説で慎重に記述すること。
    - 特に等辺（二等辺）の性質を持つかどうか（例：$45^\circ$ は対辺＝隣辺）は明記すること。

4.  **文化的・教育的な背景**:
    - 米国・英国の教育では、三角関数を「関数 $f(x)$ の表式」として早期に扱う傾向にある（例：正弦波 $y = \sin x$）。
    - 日本の教育では、幾何学的な「直角三角形の比」として導入される。両者を意識し、比率としての正確さを第一に考慮しなさい。

### 例：変種の問題生成
**元問題**: 斜辺 5 の直角三角形、$\theta = 53^\circ$（$\tan \theta \approx 4/3$ など）
**変種**: 斜辺が 10 の直角三角形、$\theta = 30^\circ$
- **生成**: 「斜辺 10」から直角三角形を構成し、$\theta=30^\circ$ の対辺 $a$ を求めなさい。
- **生成ルール**: 「$\theta = 30^\circ$ の場合、斜辺を $L$ に換算し、$a = L/2$ を求めなさい」という問題文を生成する。

## 補足ノート (Notes)

- **定義の拡張**: 三角関数は現在、単に「直角三角形の比」ではなく、座標平面（単位円）における定義へ拡張されます。$\theta = 0^\circ$ または $90^\circ$（$\pi/2$）の場合の定義は、幾何学的には斜辺または対辺が存在しないように解釈できず、定義は通常「鋭角 $0 < \theta < \pi/2$」に限定されています。しかし、関数理論においては角度が $90^\circ$ 以上（第2・第3・第4象限）でも定義可能（三角関数の一般化）となります。
- **名称の由来**:
  - **Sine (正弦)**: 古代インド語からギリシャ語を経て「chord（弦）」との関係から派生し、現在の用語になりました。
  - **Cosine (余弦)**: 補角（余角）の正弦値と一致するため「余弦」と呼ばれています。
  - **Tangent (正接)**: 接線（接しそうな線）が幾何的に接する関係から来ています。単位円における接線の長さと関連があります。

- **単位**: 角度の単位として、度数法（$^\circ$）とラジアン法（$\text{rad}$）があります。この教材では度数法（$^\circ$）が中心です。ラジアン（円周率 $\pi$ の関係を考慮）も必要に応じて触れます。

- **用語**: 三角形における対辺（Opposite）、隣辺（Adjacent）、斜辺（Hypotenuse）は、どの三角形でも固定された役割を担います。
  - $\sin \theta$ は対辺/斜辺、$\cos \theta$ は隣辺/斜辺、$\tan \/geometric}$.
