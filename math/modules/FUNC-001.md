## 概念説明

関数とは、2 つの集合 X と Y が与えられたとき、X の要素のつづり（入力）に対して、Y の要素のつづり（出力）を 1 つだけきまりさせる規則のことを指します。これは、入力と出力の関係を規則によって定義する数学的な対象であり、現代数学の基礎をなす重要な概念です。

関数を表す記号には、f(x) や g(x) などが用いられ、これらを『f で x が定義された値』あるいは『f(x) 』と読みます。x を『変数』または『入力変数』と呼び、y = f(x) とおく場合、y を『関数 f の x における値』あるいは『f(x) 』と呼びます。

重要なのは、関数の記号 f(x) と x の役割の違いに注意することです。f(x) は関数全体の表記であり、x は具体的な入力値を指します。例えば、f を関数とするとき、f(x) は関数 f が x に与える値を表します。これを、ある規則で与えられた関数の定義式と呼びます。

**定義域**とは、関数が適用可能な入力値の集合を指し、すべての可能な x の値の範囲を表します。**値域**とは、関数が実際にとる値の集合を指し、すべての可能な f(x) の値の範囲を表します。定義域は問題文で明示された場合や、式が意味を持つ範囲で自然に決まる場合があり、値域は定義式から実際に得られる最大の範囲を指定します。

関数の定義式 y = f(x) または f: x → f(x) を書くことで、入力 x の値に対してどのように値が決まるかを明確に示します。この表記は、入力から出力への規則を視覚的に表すものであり、代数式の操作だけでなく概念的な理解を深めるために重要です。

関数には線形関数、二次関数、三角関数など多くの種類があり、それぞれ特有の性質を持っています。しかし、それらに関わらず、すべての関数には定義域と値域という 2 つの集合が存在し、これらを明確に理解することが必須の基礎事項となります。

## 例題

### 例題 1

関数 f を、f(x) = 2x + 1 とする。x = 3 のとき f(x) の値を求めよ。

**解答:**

まず、与えられた関数の定義式 f(x) = 2x + 1 において、x に具体的な値 3 を代入します。

f(3) = 2 × 3 + 1 = 6 + 1 = 7

したがって、x = 3 のとき f(x) = 7 となります。

つまり、f(3) = 7 と書くことができます。

### 例題 2

関数 g を、g(x) = x² - 4x + 5 とする。この関数の定義域と値域を調べよ。ただし、定義域は実数全体とします。

**解答:**

定義域：問題文より「実数全体」とあるので、定義域は ℝ (すべての実数の集合) となります。

値域：この式は二次関数です。二次関数 y = ax² + bx + c のグラフは放物線であり、a < 0 の場合は上に凸、a > 0 の場合は下に凸です。

a = 1 > 0 なので、下に凸の放物線です。頂点における値が最小値となり、それ以上すべての値をとります。

頂点の x 座標は、x = -b/(2a) = 4/(2×1) = 2 となります。

頂点における y 座標（最小値）を計算します：
g(2) = 2² - 4×2 + 5 = 4 - 8 + 5 = 1

したがって、最小値は 1 であり、それより大きなすべての値をとります。

値域は [1, ∞) となります。

## 練習問題

### 問題 1

関数 h を、h(x) = 3x - 7 とする。以下の問いに答えよ。
(1) h(5) の値を求めよ。
(2) x = -2 のとき h(x) の値を求めよ。
(3) h(x) = 2 のとき x の値を求めよ。

**解答:**
(1) h(5) = 3×5 - 7 = 15 - 7 = 8

(2) h(-2) = 3×(-2) - 7 = -6 - 7 = -13

(3) 3x - 7 = 2 より、3x = 9、x = 3

**採点基準:**
(1) 正しい計算式と答え（8）
(2) 正しい計算式と答え（-13）
(3) 正しい方程式の解き方と答え（3）
誤答例：定義式の代入ミス、符号の置き換えミス

### 問題 2

関数 k を、k(x) = (x + 1) / (x - 3) とする。
以下の問いに答えよ。
(1) この関数の定義域を求めよ。
(2) x = 0 のとき k(x) の値を求めよ。
(3) k(x) = 0 のとき x の値を求めよ。

**解答:**
定義域：分母が 0 にならないことが必要なので、x - 3 ≠ 0、つまり x ≠ 3 です。
定義域は ℝ - {3} と表します。

k(0) = (0 + 1) / (0 - 3) = 1 / -3 = -1/3

k(x) = 0 より、分子が 0 になればよい：x + 1 = 0、x = -1
ただし、x = -1 は定義域に含まれるので有効です。

**採点基準:**
(1) 定義域の説明と記述（分母が 0 にならない条件）
(2) 代入計算の正しさと値（-1/3）
(3) 分子=0 の条件の導出と答え（-1）
誤答例：定義域を ℝ とする、分数計算の誤り、分子を分母と混同

### 問題 3

関数 m を、m(x) = 2x² - 8x + 10 とする。以下の問いに答えよ。
(1) 定義域を求めよ（実数全体とします）。
(2) 値域を求めよ。
(3) m(x) の最小値を求めよ。

**解答:**
(1) 定義域は実数全体 ℝ です。

(2) 値域を求める：頂点における値が最小値です。
頂点の x 座標：x = -(-8)/(2×2) = 8/4 = 2
m(2) = 2×4 - 8×2 + 10 = 8 - 16 + 10 = 2

最小値は 2 であるので、値域は [2, ∞) です。

(3) 最小値の計算は値域の計算と同じ：m(2) = 2

**採点基準:**
(1) 定義域 ℝ の記述
(2) 値域の導出過程と記述（[2, ∞)）
(3) 最小値の計算式と答え（2）
誤答例：値域の下限計算ミス、定義域を正しく設定しない

## 問題生成ルール

線形関数の問題を作成するルール:
- 関数の定義式は f(x) = px + q の形で、p は 0 でない実数とする。
- p と q の値は、整数または 1/2, 1/3 の値を使用します。
- 入力変数 x は整数または簡単な分数を使用します。
- 出題は単一の計算や方程式解決、または値域の簡単な考察に限定します。
- 定義域の条件は、分母が 0 にならない場合を除き、通常は実数全体とします。

## 概念説明

線形関数とは、関数 f の定義式が次のように表せる関数のことを指します。
$$f(x) = ax + b$$
ここで、a は 0 でない実数であり、b も実数です。このとき、関数 f を、x における値を f(x) と表します。

この形は、x に対して直線的な変化をするものを表し、a を『傾き』、b を『切片』と呼びることが一般的です。傾き a は、x が増加したときに関数の値がどれくらい変化するかを表し、切片 b は、x = 0 のときの関数の値を表します。

### グラフと方程式

関数 $f(x) = ax + b$ のグラフは、直線になります。x 軸との交点は (0, b) で、y 軸との交点も同じです（つまり、切片）。a はその直線の傾きです。

グラフの作成では、2 点を決めることで直線が描けます。例えば、x = 0 のとき y = b、x = -b/a のとき y = 0 の点を結ぶことで、グラフを描くことができます。

**定義域と値域の考慮**:
- 通常、定義域は実数全体 ℝ とします（a = 0 以外、または分母が 0 にならない場合）。
- 値域は、a ≠ 0 の場合 ℝ となります。

## 例題

### 例題 1

関数 f を $f(x) = 2x - 3$ とする。
以下の問いに答えよ。
(1) f(x) のグラフを x 軸・y 軸との交点を指定して描け。
(2) f(0) の値を求めよ。
(3) f(x) = 0 のときの x の値を求めよ。

**解答:**
(1) 交点の座標:
x 軸との交点（f(x) = 0 のとき）:
2x - 3 = 0 → 2x = 3 → x = 3/2 → 交点 (3/2, 0)
y 軸との交点（x = 0 のとき）:
f(0) = 2×0 - 3 = -3 → 交点 (0, -3)

これら 2 点 (3/2, 0) と (0, -3) を結ぶことでグラフが描けます。

(2) f(0) = -3 となります。

(3) f(x) = 0 のとき x = 3/2 です。

**グラフの視覚的説明**:
グラフは、y 軸上に -3 を示し、x 軸上に 3/2 を示す 2 点を結んだ直線になります。傾き a = 2 は、x が増加すると y も増加（右上がり）であることを意味します。

### 例題 2

関数 $f(x) = x^2 - 4x + 5$ とする。
以下の問いに答えよ。
(1) 定義域を求めよ（実数全体）。
(2) 頂点の座標を求めよ。
(3) 値域を求めよ。

**解答:**
(1) 定義域は ℝ です。

(2) 頂点の座標: 二次関数であり、頂点の x 座標は -b/(2a) で求められます。
x = 4/(2×1) = 2
f(2) = 2² - 4×2 + 5 = 4 - 8 + 5 = 1
頂点は (2, 1) です。

(3) 値域: 最小値は 1 なので、値域は [1, ∞) です。

## 概念説明

二次関数とは、二次項 x² の係数が 0 でない関数のことです。二次関数 f の一般的な形は以下の通りです。

$$f(x) = ax^2 + bx + c \quad (a \neq 0)$$

このグラフは放物線であり、a の符号によって上に凸または下に凸になります。

### グラフと頂点

頂点の x 座標は、x = -b/(2a) で求められます。頂点の y 座標は、この x を式に代入することで求められます。この頂点における値が関数の最大値または最小値となります。

### 値域の導出

- 下に凸（a > 0）: 値域は [最小値, ∞)
- 上に凸（a < 0）: 値域は (-∞, 最大値]

### 三角関数

三角関数（sin, cos, tan）は、角度の関数であり、通常は度数法または弧度法で表します。定義域は角度の集まり、値域は [-1, 1] です（tan は例外）。定義域と値域を意識することで、関数の性質を把握できます。

## 例題

### 例題 1

関数 f を $f(x) = x^2 - 6x + 9$ とする。
以下の問いに答えよ。
(1) 定義域を求めよ（実数全体）。
(2) 頂点の座標と、最小値を求めよ。
(3) f(x) = 0 のときの x の値を求めよ。

**解答:**
(1) 定義域は ℝ です。

(2) 頂点の x 座標:
x = 6/(2×1) = 3
f(3) = 3² - 6×3 + 9 = 9 - 18 + 9 = 0
最小値は 0 で、頂点は (3, 0) です。

(3) f(x) = 0 のとき、x = 3 です。

**グラフの視覚的説明**:
頂点が x 軸上にあり、グラフは x 軸を接しています。a > 0 なので下に凸です。

### 例題 2

関数 f を $f(x) = -x^2 + 4x - 3$ とする。
以下の問いに上に凸か下に凸かを指定し、最小/最大値を求めよ。
(1) a の符号を判断し、凸性の記述。
(1) 頂点の座標と最大値を求めよ。
(2) 値域を求めよ。

**解答:**
(1) a = -1 < 0 なので、上に凸です。

(2) 頂点の x 座標: x = -4/(2×(-1)) = 2
最大値: f(2) = -2² + 4×2 - 3 = -4 + 8 - 3 = 1
頂点は (2, 1) です。

(3) 値域は (-∞, 1] です。

**グラフの視覚的説明**:
頂点が (2, 1) で、a < 0 なので上に凸の放物線を描きます。

### 定義域・値域の制約

このテキストは、定義域や値域について言及しています。定義域は実数全体 ℝ とするのが一般的ですが、分母が 0 にならないなど、分岐が必要です。値域は a の符号によって決定されます。

このテキストは、定義域と値域の概念を整理し、次章ではより複雑な複素数や関数、あるいは微積分を扱う内容へと進展します。
