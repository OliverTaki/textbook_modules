## 概念説明

関数とは、ある集合から別の集合へ要素を「一意的に」対応付ける規則のことです。具体的には、入力された値が1つ決まれば、必ず出てくる結果が1つだけの関係を指します。

### 関数の基本構造

関数は一般的に「f(x)」といった表記を使います。ここで「f」は関数そのもの（関数名）を表し、「x」は入力変数を意味します。例えば f(x) = 2x + 1 という式は、「x に対して 2 倍して 1 を足したものが結果となる」という規則を表しています。

**重要な原則:**
1 関数における各 x に対して、y (または f(x)) の値はただ 1 つしか決まらない。例えば x = 3 のとき y = 7 かつ y = 9 は同時に成立させることはできない。

### 定義域（Domain）

定義域とは、関数に対して意味のある入力値が許容される x の範囲のことです。現実的な文脈では「物理的に可能ない値」と「数学的に定義できない値」が存在します。

数学的な制限例:
- 分母が 0 になる値（例：分母に x がある場合、x ≠ 0）
- 平方根の中身が負になる値（偶数次根の場合）
- 対数の中身が負や 0 になる値
- 三角関数の逆関数で定義域を指定する場合など

例えば関数 f(x) = 1/x は、x = 0 の値は定義できないため、定義域は全体実数から 0 を抜いた集合 {x | x ≠ 0} となります。

### 値域（Range）

値域とは、関数に対して実際に出てくる y の値の集合のことです。すべての x を通り抜ければ出てくる y の値の範囲を意味します。

例:
- f(x) = x² （定義域が全体実数） → 値域は {y | y ≥ 0}
- f(x) = sin(x) （定義域が全体実数） → 値域は {y | -1 ≤ y ≤ 1}

### 関係式と関数の違い

「y = x²」と「関数 f(x) = x²」は区別されます。前者は「点の集合（グラフ）」を意味し、後者は「計算規則（写像）」として理解されます。教科書や高校教育では両者の使い分けを意識する必要があります。

## 例題

### 例題 1

**問: 以下の関数の定義域を求めなさい。**

a) f(x) = √(x - 3)  
b) g(x) = 1/(x + 2)  
c) h(x) = log₂(x - 5)

---

**解答:**

**a)** √(x - 3) の下には 3 乗根でも 2 乗根でも、偶数次根の中身は負にならないようにしなければならない。

√(x - 3) が定義されるためには x - 3 ≥ 0 でなければならない。

x - 3 ≥ 0  
→ x ≥ 3

**定義域:** {x | x ≥ 3} または区間表記で [3, ∞)

---

**b)** 分母が 0 にならず、0 でなければならない。

1/(x + 2) で分母は (x + 2) なので、これが 0 にならないようにする。

x + 2 ≠ 0  
→ x ≠ -2

**定義域:** {x | x ≠ -2} または区間表記で (-∞, -2) ∪ (-2, ∞)

---

**c)** 対数関数 log₂(x - 5) において、対数の中身は正でなければならない。

x - 5 > 0  
→ x > 5

**定義域:** {x | x > 5} または区間表記で (5, ∞)

---

### 例題 2

**問: 以下の関数の値域を求めなさい。**

a) f(x) = 2x + 1 （定義域は全体実数）  
b) g(x) = x² （定義域は全体実数）  
c) h(x) = 1/x （定義域は x ≠ 0）

---

**解答:**

**a)** f(x) = 2x + 1 は一次関数の形で、傾きが 2 で y 切片が 1 の直線です。傾きが 0 ではない一次関数は定義域が全体実数の場合、値域も全体実数となります。

**値域:** {y | y ∈ ℝ} または (-∞, ∞)

---

**b)** g(x) = x² は 2 乗関数です。x がどんな実数でも、x² は常に 0 またそれ以上の非負の値になります。最小値は x = 0 の時の g(0) = 0 であります。

**値域:** {y | y ≥ 0} または [0, ∞)

---

**c)** h(x) = 1/x は反比例関数の形のものです（x ≠ 0 の場合、定義域全体から 0 を除く）。

x が 0 に近づくほど値は無限大に近づくが決して 0 にはなりません。x が無限大に近づくほど値は 0 に近づく。

**値域:** {y | y ≠ 0} または (-∞, 0) ∪ (0, ∞)

## 練習問題

### 問題 1

**以下の関数の定義域をすべて求めなさい。**

a) f(x) = √(5 - x)  
b) g(x) = 1/(x² - 4)  
c) h(x) = ln(x + 1)

**解答:**

a) √(5 - x) が定義されるためには、5 - x ≥ 0 でなければならない。

5 - x ≥ 0  
→ x ≤ 5

*定義域:* {x | x ≤ 5} または (-∞, 5]

b) 分母が 0 にならないようにする。

x² - 4 ≠ 0  
x² ≠ 4  
x ≠ ±2

*定義域:* {x | x ≠ ±2} または (-∞, -2) ∪ (-2, 2) ∪ (2, ∞)

c) 対数の中身は正でなければならない。

x + 1 > 0  
→ x > -1

*定義域:* {x | x > -1} または (-1, ∞)

---

**採点基準:**

正解には以下の要素をすべて含めること:
- 不平等式の導出過程（偶数次根の場合は≥、分母の場合は≠、対数の場合は>）
- 最終的な定義域を集合記法または区間表記のどちらで書いているか一貫性
- 集合記法の場合、「全体実数」という表現の使い分け（必要なら明示）
- 区間表記の場合、括弧の種類（[ ] は境界値を含まない、( ) は境界値を含む）

### 問題 2

**以下の関数の値域をすべて求めなさい。**

a) f(x) = -x² + 2x + 1 （定義域は全体実数）  
b) g(x) = arctan(x) （定義域は全体実数）  
c) h(x) = e^x

**解答:**

a) 放物線 f(x) = -x² + 2x + 1 の頂点を求めると

x 座標の頂点公式: x = -b/(2a) = -2/(2×(-1)) = 1  
このとき y 座標: f(1) = -(1)² + 2(1) + 1 = -1 + 2 + 1 = 2

二次関数 a = -1 < 0 なので最大値が存在し、値域は [2, ∞) の逆であり [2, ∞) の最大値 2 までで上から制限されている。

実際、a < 0 であるため放物線は上に開くのではなく下に開いている。つまり最大値が存在し、すべての x に対して y ≤ 2 となります。

*値域:* {y | y ≤ 2} または (-∞, 2]

---

b) arctan(x) は逆正接関数。定義域は全体実数ですが、値域は -π/2 より大きく π/2 より小さい実数となります。

*値域:* {y | -π/2 < y < π/2} または (-π/2, π/2)

---

c) e^x は指数関数。e^x はどんな x を代入しても常に正の値しか生じず、限りなく 0 に近いが決して 0 にはならない。

*値域:* {y | y > 0} または (0, ∞)

---

**採点基準:**

正解には以下の要素を関数の性質（増減や頂点計算）と最終結果をすべて含める:
- 頂点計算や関数の性質の説明
- 漸近線などの性質の説明（例：反比例関数の 0 付近の挙動）
