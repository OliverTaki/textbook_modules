# 関数の導入

## 1. コンセプト

数学において「$f$」や「$g$」といった記号は、変数 $x$ を独立変数、$y$ を従属変数として $y$ を与える特定の規則を指します。これを「関数」と呼びます。この概念は、集合論における記号の「写像（mappa）」として記述されます。

具体的には、$x$ の集合（定義域）から、$y$ の集合（値域）への対応関係を定義します。この対応関係が「一意に」（1 対 1 または 1 対多でないが、各 $x$ に対して $y$ は 1 つだけ）決まっている必要があります。これは、代数方程式や解析学の基礎となります。

## 2. 定義と表記

関数は、$x$ に対して $y$ がどのように計算されるかを記述します。例えば、$y = 2x$ は「$x$ の 2 倍」を定義します。これを「線形関数（linear function）」と呼びます。

また、$x$ の関数である $y$ は、$y = f(x)$ のように書かれます。ここで $f$ は関数の名前で、$(x)$ は入力値を意味します。これは「$f$ が $x$ を入力して値を出力する」という写像のプロセスを表します。

**集合論的な定義の要点**:
- 定義域（domain）: $x$ の集合。
- 値域（range）: $y$ の集合。
- 写像（mapping）: 定義域から値域への対応。

数学的には、$x$ の集合を $A$、$y$ の集合を $B$ とし、その写像を $f: A \rightarrow B$ と記します。

## 3. 例題

以下の関数について、指定された値を計算してください。

**例題 1**: $f(x) = x^2$ のとき、$f(3)$ を計算せよ。
**解答**: $\ 3^2 = 9$。したがって $f(3) = 9$。

**例題 2**: $g(x) = 2x + 5$ のとき、$g(0)$ を計算せよ。
**解答**: $g(0) = 2\cdot 0 + 5 = 5$。

**例題 3**: 次式は関数を定義するものか判定せよ。
$$ h = \begin{cases} 0 & x < 0 \\ 1 & x \ge 0 \end{cases} $$
**解答**: 各 $x$ に対して $h$ が決定されるため、関数である。

## 4. 問題

以下の問題を解き、解答を記述せよ。計算過程は簡潔に記述すること。

**問題 1**: $f(x) = 3x + 4$ について、各々の値を求めなさい。
1. $f(-1)$
2. $f(-5)$
3. $f(13.7)$

**解答 1**:
1. $f(-1) = 3(-1) + 4 = -3 + 4 = 1$。
2. $f(-5) = 3(-5) + 4 = -15 + 4 = -11$。
3. $f(13.7) = 3(13.7) + 4 = 41.1 + 4 = 45.1$。

**問題 2**: 次の表から $f(x)$ の式を求めなさい。
| $x$ | $y$ |
|:---:|:---:|
| 0 | 1 |
| 1 | 3 |
| 2 | 5 |
| 3 | 7 |

**解答 2**: $x$ が 1 増えると $y$ が 2 増えるため、係数は 2。$y = 2x + k$ の形。
$x=0$ のとき $y=1$ より $k=1$。したがって、$f(x) = 2x + 1$。

**問題 3**: $f(x) = x^2$ のグラフ上に、点 $(2, y)$ がある。このとき $y$ の値はいくつか。また、その点の座標はいくらか。

**解答 3**: $x=2$ のとき、$y = 2^2 = 4$。したがって、点は $(2, 4)$。

**問題 4**: 定数項なしの一次関数 $f(x) = x$ と、関数 $g(x) = 2f(x)$ を考えよ。
1. $g(x)$ の式は何か。
2. $g(4)$ の値はいくらか。
3. $g(x) = 4$ のとき $x$ の値はいくらか。

**解答 4**:
1. $g(x) = 2x$。
2. $g(4) = 2 \cdot 4 = 8$。
3. $2x = 4$ より $x = 2$。

**問題 5**: 関数 $f(x) = x^2 - 4$ について、定義域を $x \in \{-2, -1, 0, 1, 2\}$ のとき、値域を求めよ。
**解答 5**:
- $x=-2: 4-4=0$
- $x=-1: 1-4=-3$
- $x=0: 0-4=-4$
- $x=1: 1-4=-3$
- $x=2: 4-4=0$
値は $\{ -4, -3, 0 \}$。
したがって、値域は $\{-4, -3, 0\}$。

**問題 6**: $x \in \{0, 1, 2\}$. この場合、$f(x) = 2x + 3$ について、$f(x)$ の値の総和はいくらか。
**解答 6**:
$f(0) = 3$
$f(1) = 5$
$f(2) = 7$
総和 = $3 + 5 + 7 = 15$。

**問題 7**: 関数 $f, g$ において、$f(x) = -x^2$, $g(x) = x^3$. $\ x \in \{-1, 0, 1\}$ の場合、$f(x) + g(x)$ の値をそれぞれ計算せよ。
**解答 7**:
- $x=-1$: $f(-1)=-1, g(-1)=-1, sum=-2$.
- $x=0$: $f(0)=0, g(0)=0, sum=0$.
- $x=1$: $f(1)=-1, g(1)=1, sum=0$.

**問題 8**: 関数 $f(x) = x^2$ のグラフの $x$ 軸との交点（定義域を $[-3, 3]$ とする）を求めなさい。
**解答 8**: $x^2 = 0 \Rightarrow x = 0$. 交点は原点 $(0,0)$ のみ。
（補足: 定義域の端点は $f(-3)=9, f(3)=9$）。

**問題 9**: $h(x) = |x|$ のとき、$h(3)$ を求めよ。
**解答 9**: $h(3) = |3| = 3$。

**問題 10**: $k(x) = |x|$ のグラフについて、$x < 0$ の場合の式の簡潔な記述をせよ。
**解答 10**: $x < 0$ の場合、$k(x) = -x$。

**問題 11**: 次の関数の定義域と値域を特定せよ。$y = x^2$ かつ $x \in \mathbb{R}$（実数全体）。
**解答 11**: 定義域 $\mathbb{R}$、値域 $[0, \infty)$。
（補足: $x^2$ は常に非負。最小値 $0$ あり、無限大に発散）。
