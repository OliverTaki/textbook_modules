## 概念説明

函数とは、入力された値に対して一意に出る出力の規則を記号化した数学的な対象です。函数は通常 f(x) という記号で表され、これは「f という函数に対する x という入力に対して得られる出力」を意味します。

函数の核心は、入力と出力との間の規則的対応関係にあります。この对应关系は、数表・グラフ・式・記述など様々な形で表現できますが、数学的作業では式を用いて定義することが一般的です。

函数 f(x) の中身は必ずしも文字 x のみとは限りません。例えば f(x+1) や f(x)² のように、式全体が函数名に付随する形で現れることもあります。ここで重要なのは、「函数というものは規則そのもの」であり、「入力された値」を「出力した値」に変換する機械と同様のものとして捉えることです。

函数の性質を理解するには、定義域と値域の概念を知ることが不可欠です。定義域とは、函数に対して意味を持つ入力の集合を指します。例えば、分母が 0 になるとする場合や、平方根の中身が負になる場合など、函数が定義できない入力があることを意味します。値域とは、函数によって得られる全ての出力値の集合で、定義域を函数に処理した結果として生じる範囲を表します。

函数は単一の入力に対して単一の出力しか許さず、2 つ異なる入力で同じ出力が得られることはあり得ても、1 つの入力で 2 つ異なる出力を得ることは許されません。これが函数を関数と呼ぶべき理由であり、集合論では「対応」として厳密に定義されます。

函数の等式は y=f(x) の形で表されることが一般的です。このとき、y 軸方向に定数を加減することは函数のグラフの上下翻訳に相当します。例えば f(x) のグラフ上に 2 を加える f(x)+2 は、元のグラフ上に 2 個下のすべての点に対し、さらに 2 分の上の位置に点を移動させる変換を意味します。

函数の合成函数 f(g(x)) は、入力を g で処理してからそれを f に渡すという 2 段階の処理を意味します。これは計算機科学におけるパイプライン処理や、データ処理における段階的変換と対応する考え方です。

函数の逆函数は、函数を「後戻りさせる」変換を意味します。逆函数が存在するためには、function が全単射である必要があります。しかし、単一の函数に対して逆函数が 1 つに定まることを保証するためには、函数の定義域や値域を適切に制限する必要がある場合があります。

函数の概念は数学のあらゆる分野の基礎であり、微分積分学における函数の連続性・微分可能性の議論、線形代数における変換行列の理解、確率論における確率密度函数的考え方など、全ての分野で函数という概念には戻ってくる必要があります。

## 例題

### 例題 1

関数 f(x) = x² - 2x + 3 の定義域と値域を求めなさい。また、グラフの頂点座標を求めなさい。

**解答:**

この函数は一次式から一次式を引いた形の二次函数である。係数を整理すると、f(x) = x² + (-2)x + 3 です。頂点の x 座標は -b / (2a) によって求められます。ただし、a=1、b=-2 なので、頂点の x 座標は -(-2) / (2×1) = 2 / 2 = 1 と求められます。

x=1 のとき、f(1) = 1² - 2(1) + 3 = 1 - 2 + 3 = 2 です。したがって、頂点の座標は (1, 2) です。

定義域について考えます。この函数は二次関数で、多項函数なので全ての実数 x に対して定義されます。定義域は R で表されます。

値域について考えると、a=1 となり正の係数を持つため、パラボラは上に凸ではなく上に開いており、頂点が最小点になります。頂点の y 座標は 2 で、これが最小値となるため、値域は [2, ∞) です。

**採点基準:**
- 定義域が R と明示されていること（10 点）
- 値域が [2, ∞) としており、最小値 2 が示されていること（10 点）
- 頂点座標 (1, 2) が正しいこと（10 点）
- 計算過程が示されていること（20 点）
- 合計 50 点満点、70 点が上記解答基準を満たす

### 例題 2

定義域が [0, 4] の函数 f(x) = √(x-1) + 2 について、値域を求めなさい。また、x=3 のとき、f(3) の値を計算しなさい。

**解答:**

まず x が何の値でもよいのかを考えます。平方根の中身 x-1 は常に 0 以上でなければならないため、x≥1 でなければならないことがわかります。与えられた定義域 [0, 4] と合わせると、実際の有効な入力範囲は [1, 4] となります。

この場合、x=1 のとき f(x)=2 となり、x=4 のとき f(x)=√3+2 です。平方根函数は単調増加であるため、値域は [2, √3+2] です。

x=3 のとき、f(3) = √(3-1) + 2 = √2 + 2 と計算されます。これが正確な値です。

**採点基準:**
- 平方根の中身が非負なので x≥1 と認識できること（15 点）
- 有効な定義域が[1, 4]としており、元の定義域と区別できること（15 点）
- 値域が[2, √3+2]として、平方根単調性を考慮していること（15 点）
- f(3) の計算が √2 + 2 と正しいこと（15 点）
- 合計 60 点満点、80 点が上記解答基準を満たす

## 練習問題

### 問題 1

関数 f(x) = 2x - 5 について、以下の各入力のときの出力を求めなさい。

（ア）f(0) の値

（イ）f(3) の値

（ウ）f(-2) の値

**解答:**
（ア）f(0) = 2(0) - 5 = -5

（イ）f(3) = 2(3) - 5 = 6 - 5 = 1

（ウ）f(-2) = 2(-2) - 5 = -4 - 5 = -9

**採点基準:**
- （ア）で -5 として計算過程が示されていること
- （イ）で 1 として計算過程が示されていること
- （ウ）で -9 として計算過程が示されていること
- 誤りがないこと、計算ミスがないこと

### 問題 2

関数 g(x) = 1 / (x - 3) について、定義域と値域を求めなさい。また、この函数が定義できない x の値を特定しなさい。

**解答:**
この函数において分母が 0 にならないようにするため、x-3≠0 でなければなりません。したがって、x≠3 となり、定義域はすべての実数から 3 を除いたもの、すなわち x∈R, x≠3 です。

値域について考えると、g(x) がどのような値をとるのでしょうか。g(x) = 1 / (x-3) であり、分母が無限大に近づけば g(x) は 0 に近づきますが、0 にはなりません。なぜなら、1/anything が 0 になることはありません。また、分母がどのような値をとっても g(x) は 0 にはなりません。

したがって、値域はすべての実数から 0 を除いたもの、すなわち y∈R, y≠0 です。

函数が定義できないのは x=3 のときです。このとき分母が 0 となり、数学的に未定義となります。

**採点基準:**
- 定義域が x≠3 と示されていること
- 値域が 0 を除いた実数であることが示されていること
- x=3 で定義できない理由として「分母が 0 になるから」と明記されていること
- 値域の 0 が除かれる理由が説明されていること
