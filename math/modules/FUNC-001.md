## 概念説明

関数とは、特定の集合である定義域（入力）の要素が、規則に従って対応する一つだけの値（出力）を持つ写像のことです。簡単にいうと、「入力をひとつ与えると、常に同じ方法で一つの答えが得られる関係」が関数です。

例えば、y = 2x + 1 という式を考えてみます。ここで x に 1 を代入すると y = 3、x に 2 を代入すると y = 5 となります。このとき、x は「定義域」の要素で、y は「値域」の要素です。重要なのは、定義域から同じ値を選んでも、常に同じ y の値が得られるということです。もし x=3 で y が 5 と 6 の両方となってしまうような関係なら、それは関数ではなくなります。

関数を表すには、f(x) などと書くこともよくあります。ここで f は関数の名前、x は定義域の要素、f(x) は x に対して返される値を表します。定義域は、この関数において意味をなすことができる入力値の全体を指し、値域は実際に得られる出力値の全体を指します。定義域が与えられることを前提にすれば、値域は自動的に決まりますが、場合によっては指定されることもあります。

関数の性質を議論する上で、定義域と値域を理解することは極めて重要です。なぜなら、多くの場合「計算できるから何でも良い」と誤解されるためです。例えば分母に0が入る式、根号に負の値が入る式の類は、値が定義できないため、それらの値は定義域に含まれません。このような制約を理解せずに式だけを操作するのは、数学の思考を誤らせる危険な態度です。したがって、定義域の確認は常に最初に考えるべきステップです。

## 例題

### 例題 1

関数 f(x) = x² + 2x − 3 の場合を考える。

**【問題】**
次のことを示せ。
(1) この関数の定義域を求めよ。
(2) x について整理した式 f(x) を用いて、定義域が実数全体の場合に、値域が何であるかを示せ。

**【解答】**

(1) 
この関数 f(x) = x² + 2x − 3 は、多項式関数です。多項式関数は任意の実数に対して定義されています。したがって、この関数の定義域は「実数全体 x | x ∈ R」となります。

(2)
定義域が実数全体の場合、二次関数 y = x² + 2x − 3 を考えます。このグラフは放物線であり、二次項の係数が正（1）であるため、下に凸の放物線を描きます。

頂点の x 座標を求めます：x = −b/(2a) = −2/(2×1) = −1

頂点の x = −1 のときの y 値：f(−1) = (−1)² + 2(−1) − 3 = 1 − 2 − 3 = −4

下に凸の放物線において、頂点の y 座標は最小値です。したがって、値域は「−4 以上のすべての実数 y | y ≥ −4」となります。

### 例題 2

関数 g(x) = √(x − 3) の場合を考える。

**【問題】**
次のことを示せ。
(1) この関数の定義域を求め、不等式で表現せよ。
(2) この関数の値域を求め、不等式で表現せよ。

**【解答】**

(1) 根号の中身が負の数になることはあり得ません。したがって、根号の中身 x − 3 ≥ 0 でなければなりません。これを解くと：

x ≥ 3

よって、定義域は「x ≥ 3」となります。集合表記では {x | x ≥ 3} または [3, ∞) と記述します。

(2) 
定義域が x ≥ 3 のとき、根号の中身 x − 3 ≥ 0 です。根号の中身が 0 のとき、g(x) = 0 となり、これが最小値です。x → ∞ のとき、g(x) → ∞ です。

つまり、値域は「0 以上のすべての実数 y | y ≥ 0」となります。集合表記では {y | y ≥ 0} または [0, ∞) と記述します。

## 練習問題

### 問題 1

関数 h(x) = 1/(x − 4) のとき、次のことを回答せよ。

**問題文：**
(1) この関数の定義域を求めなさい。
(2) この関数の値域を求めなさい。
(3) x = 4 のときに式が定義できない理由は何か説明せよ。

**解答：**
(1) 分母は0になれません。したがって x − 4 ≠ 0、つまり x ≠ 4 です。定義域は {x | x ≠ 4} または (−∞, 4) ∪ (4, ∞) と表します。

(2) 分母がどのような値であっても、結果は 0 になりません。h(x) = 0 を満たす x は存在しないため、値域は {y | y ≠ 0} または (−∞, 0) ∪ (0, ∞) と表します。

(3) x = 4 のとき、分母が 0 になり、0 で割ることは数学的に意味を持ちません（未定義）。したがって、この関数は x = 4 で定義されません。

**採点基準：**
- 定義域：x ≠ 4 と正しい表現で記述していることが必須
- 値域：y ≠ 0 と正しい表現で記述していることが必須
- 理由：「分母が0になるため」または類似の誤解を招かない言葉で記述していることが必須
- 全問正解：30点、部分正解：10〜20点、不正解：0点

### 問題 2

関数 k(x) = x³ のとき、次のことを回答せよ。

**問題文：**
(1) この関数の定義域と値域を求めなさい。
(2) この関数是一对対応（一一の対応）であることを説明して、逆関数が存在する理由を述べよ。
(3) この関数を反転させる（逆関数として考える）と、どのような関係式が得られるか。

**解答：**
(1) カube（三乗）関数は、実数の任意の値に対して定義可能です。定義域は実数全体 {x | x ∈ R} に、値域も実数全体 {y | y ∈ R} です。

(2) この関数 f(x) = x³ は、区間 [a, b] においてstrictly increasing（厳密に増加）関数です。つまり、x₁ < x₂ のときに always f(x₁) < f(x₂) が成立します。したがって、定義域から一つを選べば、値域で唯一の対応物が得られます。逆関数 k⁻¹(y) = ³√y（y の立方根）が存在します。

(3) 逆関数 g(y) を求めるため、y = x³ と置換し、x について解くと、x = ³√y が得られます。したがって、逆関数は g(y) = ³√y と書けます。変数の記号を y を x に変更すると h(x) = ³√x と表せます。

**採点基準：**
- 定義域と値域：実数全体と正しく記述していることが必須（15点）
- 一对一対応の説明：increasing nature や graph の傾きを説明していることが必須（10点）
- 逆関数の式：適切に導出されていることが必須（10点）
- 全問正解：35点、部分正解：10〜30点、不正解：0点

### 問題 3

関数 m(x) = −2|x − 1| + 3 のとき、次のことを回答せよ。

**問題文：**
(1) この関数の定義域を求めなさい。
(2) 絶対値関数を用いたこの関数のグラフの頂点座標を求めなさい。
(3) この関数の値域を求めなさい。
(4) この関数が最大値を持たない理由（または最大値を持つ理由）を説明し、実際の最大値を求めなさい。

**解答：**
(1) 絶対値関数は任意の実数に対して定義可能です。したがって、定義域は実数全体 {x | x ∈ R} です。

(2) 絶対値関数 |x − 1| の最小値は x = 1 のとき 0 です。このとき m(1) = −2|0| + 3 = 3 です。頂点は (1, 3) であり、これは最大値の点です。

(3) 最大値が 3 なので、−2|x−1| は常に 0 以下であり、−2|x−1| ≤ 0 です。すると、m(x) ≤ 3 となります。また、x → ∞ のとき |x−1| → ∞ となり、−2|x−1| → −∞ です。したがって、値域は {y | y ≤ 3} と表します。

(3) [注意：元の文書には2つの(3)が含まれています。ここでは最初のものを用いて、(4)は次の文で修正されます]

(4) この関数は最大値を持ちます。絶対値の中身が大きい程、−2|x−1| は小さくなるため、全体としては最大値を持ちます。実際に計算すると、頂点における値が3であり、これが最大値となります。

**採点基準：**
- 定義域：{x | x ∈ R} などと正しく記述していることが必須
- 頂点：(1, 3) など正しく計算していることが必須
- 値域：y ≤ 3 と正しい表現で記述していることが必須
- 最大値に関する説明：「−2|x−1| ≤ 0」または「下に凸」という誤解を招かない言葉で記述していることが必須
- 全問正解：35点、採点基準を満たさない：10〜30点、不正解：scoring not available/not pass

## 補足

1. 定義域と値域の概念を理解していないと、計算が誤る危険な式（例：√(負の値) や 0 除算）に遭遇します。
   数学的思考は、計算のみで終わるのではなく、式の意味と制約を常に意識する必要があります。

2. 絶対値付き関数や絶対値の性質を理解していないと、二次関数や分式関数とは異なるグラフの形や値域を見誤ります。
   V 字状のグラフや最大値・最小値について注意が必要です。

## 補足 2

1. 反転させる（逆関数）操作は、関係式 y = f(x) から x = f⁻¹(y) への変換に他なりません。
   定義域と値域が互いに交換されることに注意が必要です。
   逆関数が存在するためには、原本関数が一一の対応（strictly increasing/decreasing）する必要があります。
   
2. 絶対値関数 |x| は、x ≥ 0 のとき x、x < m 0 のとき −x となります。
   その性質を利用することで、x の絶対値の符号を特定できます。

## 補足 3

1. 反転させる（逆関数）操作は、関係式 y = f(x) から x = fnotes[^1][^2] に他なりません。
   定義域と値域が互いに交換されることに注意が必要です。
   逆関数が存在するためには、原本関数が一一の対応（strictly increasing/de practicing）する必要があります。

[^1]: 「strictly increasing」の意味は「x が大きくなればなるほど y も大きくなる」ことです。
    二次関数の場合は、定義域を区間ごとに区別すると、右半分は単調増加、左半分は単調減少になります。
