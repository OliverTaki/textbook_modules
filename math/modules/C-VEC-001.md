## 概念説明

**ベクトルとは何か**

ベクトルとは、大きさ（成分）と方向の両方を持つ量を表す数学的な概念です。物理学的には「速度」「力」「加速度」などが典型的なベクトルの例として挙げられます。これに対し、大きさだけの「距離」「質量」などはスカラー量と呼ばれます。

ベクトルは一般的に小文字の大文字付きまたは矢印記号で表されます。例えば\(\vec{v}\)、または\(\vec{AB}\)（A から B へ向かうベクトル）という表記があります。図中的には、有向線分で表され、矢印が方向を示します。

**ベクトルの成分表示**

平面ベクトルでは、直角座標系を用いて\(\vec{v}=(a,b)\)と表記することが一般的です。ここで\(a\)はx 方向への成分（横方向）、\(b\)はy 方向への成分（縦方向）を意味します。\(\vec{v}\)の大きさ（長さを）は\(|\vec{v}|=\sqrt{a^2+b^2}\)で計算されます。

**ベクトルの演算**

ベクトルの基本演算には以下が挙げられます：

1. **加法**：\(\vec{u}=(a_1,b_1)\)、\(\vec{v}=(a_2,b_2)\)の場合、\(\vec{u}+\vec{v}=(a_1+a_2,b_1+b_2)\)
2. **減法**：\(\vec{u}-\vec{v}=(a_1-a_2,b_1-b_2)\)
3. **数乗**：実数\(k\)を用いて\(k\vec{u}=(ka_1,ka_2)\)
4. **内積**：\(\vec{u}\cdot\vec{v}=a_1a_2+b_1b_2\)（スカラー値）
5. **外積**：3 次元ベクトルでは\(\vec{u}\times\vec{v}\)（ベクトル値）

**単方向単位ベクトル**

x 方向単一成分を持つベクトルを\(\vec{i}=(1,0)\)、y 方向単一成分を持つベクトルを\(\vec{j}=(0,1)\)と表します。これらを用いて、任意のベクトル\(\vec{v}=(a,b)\)は\(\vec{v}=a\vec{i}+b\vec{j}\)と表せます。

この形式はベクトル演算を簡略化し、理解を助けるために重要です。特に物理や工学の分野において、成分表示を基にした計算は頻繁に行われます。

## 例題

### 例題 1

**問題**：ベクトル\(\vec{u}=(3,5)\)および\(\vec{v}=(1,-2)\)について、以下の計算を行ってください。

1. \(\vec{u}+\vec{v}\)
2. \(\vec{u}-\vec{v}\)
3. \(2\vec{u}-\vec{v}\)
4. \(\vec{u}\cdot\vec{v}\)

**解答**：

1. \(\vec{u}+\vec{v}=(3+1,5+(-2))=(4,3)\)
   成分ごとの足算です。x 成分：3+1=4、y 成分：5-2=3

2. \(\vec{u}-\vec{v}=(3-1,5-(-2))=(2,7)\)
   成分ごとの引き算です。x 成分：3-1=2、y 成分：5-(-2)=7（負の符号に注意）

3. \(2\vec{u}-\vec{v}=2(3,5)-(1,-2)=(6,10)-(1,-2)=(5,12)\)
   まず\(2\vec{u}=(6,10)\)とし、さらに\(\vec{v}\)を引く計算です。
   x 成分：6-1=5、y 成分：10-(-2)=12

4. \(\vec{u}\cdot\vec{v}=3\times1+5\times(-2)=3-10=-7\)
   内積は成分ごとの積の総和です。注意：この結果はスカラー（数）です。

**解説**：
この計算はベクトル演算の基本です。成分表示を用いることで、複雑な幾何的な操作を単純な足し引きに変形可能になります。特に負の成分を扱う際、符号誤りに注意が必要です。

### 例題 2

**問題**：\(\vec{OA}=(2,-3)\)、\(\vec{OB}=(5,-1)\)とする。以下の点を求めなさい。

1. ベクトル\(\vec{AB}\)を求めなさい。
2. \(|\vec{AB}|\)（直線AB の長さ）を計算しなさい。

**解答**：

1. まず、\(\vec{AB}=\vec{OB}-\vec{OA}\)です。
   \(\vec{AB}=(5-2,-1-(-3))=(3,2)\)

   幾何学的解釈：\(\vec{AB}\)はA 点からB 点へ向かうベクトルであり、B−A の差として計算できます。

2. \(|\vec{AB}|=\sqrt{3^2+2^2}=\sqrt{9+4}=\sqrt{13}\)

   大きさはピタゴラスの定理を使用して計算されます。

**解説**：
この問題では、2 点間のベクトルを表す際、終点から始点へ引くのではなく、終点を始点から引く形で表します。これは「位置ベクトル」の考え方に重要です。\(\vec{OA}\)は原点からA 点へのベクトルであり、\(\vec{AB}\)はA から B へのベクトルです。

## 練習問題

### 問題 1

\(\vec{p}=(4,7)\)、\(\vec{q}=(-2,3)\)とする。以下の計算を行なさい。

1. \(\vec{p}+\vec{q}\)
2. \(\vec{p}-\vec{q}\)
3. \(3\vec{p}-2\vec{q}\)
4. \(\vec{p}\cdot\vec{q}\)

**解答**：
1. \(\vec{p}+\vec{q}=(4+(-2),7+3)=(2,10)\)
2. \(\vec{p}-\vec{q}=(4-(-2),7-3)=(6,4)\)
3. \(3(4,7)-2(-2,3)=(12,21)-(-4,6)=(16,15)\)
4. \(\vec{p}\cdot\vec{q}=4\times(-2)+7\times3=-8+21=13\)

**採点基準**：
- 各計算の過程が正しければ OK
- 数値計算の正確さ、符号に誤りがないこと
- 内積はスカラー値であることを明確にしてあること

### 問題 2

点\((1,2)\)と点\((x,y)\)を結ぶベクトル\(\vec{v}\)について、以下が成り立つときを求めなさい。

\(\vec{v}\cdot\vec{i}=5\)（ここで\(\vec{i}=(1,0)\)）

**解答**：
\(\vec{i}=(1,0)\)で、\(\vec{v}\)の始点\((1,2)\)、終点\((x,y)\)とすると：
\(\vec{v}=\vec{v}_{終}-\vec{v}_{始}=(x-1,y-2)\)

条件より：
\(\vec{v}\cdot\vec{i}=(x-1)\times1+(y-2)\times0=x-1=5\)

したがって\(x=6\)

（y は自由変数で任意の実数に取れる）

**採点基準**：
- 位置ベクトルの定義式を理解しているか
- 内積計算が正しいか
- 条件式から x を特定できるか
