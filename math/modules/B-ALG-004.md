## 概念説明

連立方程式とは、2つ以上の変数を含んだ方程式をまとめて、すべての変数の値を同時に決定（特定させる）ための数学的な手法です。

基本的な考え方は次の通りです：

- **方程式**：「1つの式で変数間の関係を表現」
- **連立方程式**：「複数の方程式を組み合わせ、すべての変数を同時に決定」

例えば、2変数の連立方程式

1. `2x + 3y = 8`
2. `x - y = 1`

は、`x` と `y` の2つの未知数を同時に決定します。

### 代数的解法の意義

一方の式で変数を表す「置換法」と、一方の式をもう一方の式で「係数調整」する「代入法」があります。

**代入法**：
- 一方の式を変数について解く（例：`y = 2x - 4`）
- もう一方の式に代入して解けるようになる
- 結果を元の式に戻して計算を確認

**係数調整・加減法**：
- 両式の係数を調整（共通倍数）
- 一方を変形して係数をゼロにする
- 加減後、1変数に帰約

### 幾何学的解釈（文部科学省方針）

連立方程式の解は、2直線の**交点**になります。

- **解が1つ**：2直線が1点で交わる（係数比 `a1/a2 ≠ b1/b2`）
- **解がない**：2直線が平行（係数比 `a1/a2 = b1/b2`）
- **無限の解**：2直線が一致（係数比 `a1/a2 = b1/b2` で定数項も一致）

### 代入前の計算戦略

分数の式を代入する場合、**分母を共通倍数で消す**ことで計算ミスを防ぎます：

```
5x/2 - 3y/4 = 1
→ 両辺を4で掛ける → 10x - 3y = 4
```

この手法は「計算ミスを減らすための前提知識」となります。

## 例題1

### 問題

連立方程式を解け：

1. `x + y = 5`
2. `2x - y = 1`

### 解答（ステップバイステップ）

#### ステップ1：置換準備
一方の式を変数について解きます。式(1)の方が計算が簡単です。

```
式(1) から → y = 5 - x
```

#### ステップ2：代入
式(2)に `y = 5 - x` を代入します。

```
式(2): 2x - (5 - x) = 1
```

#### ステップ3：計算
```
2x - 5 + x = 1
→ 3x - 5 = 1
→ 3x = 6
→ x = 2
```

#### ステップ4：代入で他変数を求める
式(1)を `x = 2` に代入：

```
y = 5 - 2 = 3
```

#### ステップ5：結果
```
解：x = 2, y = 3
```

#### ステップ6：検算（文部科学省方針）
元の式に戻して確認：

- 式(1)：`2 + 3 = 5` ✅
- 式(2)：`2(2) - 3 = 4 - 3 = 1` ✅

#### グラフィック思考実験
- 式(1): 原点から (5, 0) と (0, 5) を結ぶ
- 式(2): (2, 1) と (-3, -2) を結ぶ

**交点**`(2, 3)`で2直線が交わっていますか？

- 式(1): `(2, 3)`→ `2 + 3 = 5` ✅
- 式(2): `(2, 3)`→ `2(2) - 3 = 1` ✅

**交点が計算結果と一致します。**

## 例題2

### 問題

連立方程式を解け：

1. `3x + 2y = 12`
2. `x - 4y = -8`

### 解答（加減法のステップバイステップ）

#### ステップ1：係数調整
加減法で消去したい変数はどちらかを決めます。

**x**を消去の場合：
- 式(1)×2 → `6x + 4y = 24`
- 式(2)に `x` の係数は 1 → `2×式(2)`で `2x - 8y = -16`

**y**を消去の場合：
- 式(1)×2 → `6x + 4y = 24`
- 式(2)×(-1) → `-x + 4y = 8`

#### ステップ2：加減計算
**y**を消去する場合：**式(1)×2 + 式(2)×(-1)**を計算：

```
(3x + 2y)=12
×2 → 6x + 4y = 24

式(2): x - 4y = -8
x → -x + 4y = -8

(6x + 4y) + (-x + 4y)  ← これは y を消去しません！```

**再計算：**

yを消去するには、係数の符号を反対にします：

```
式(1) × 2 → 6x + 4y = 24
式(2) × 1 → x - 4y = -8

(6x + 4y) + (x - 4y) = 24 + (-8)
```

#### ステップ3：計算
```
6x + x + 4y - 4y = 24 - 8
→ 7x = 16
→ x = 16/7
```

#### ステップ4：代入
式(1)に `x = 16/7` を代入：

```
3(16/7) + 2y = 12
→ 48/7 + 2y = 12
→ 2y = 12 - 48/7
→ 2y = (84 - 48)/7
→ 2y = 36/7
→ y = 18/7
```

#### ステップ5：結果
```
解: x = 16/7, y = 18/7
```

#### ステップ6：検算
- 式(1): `3(16/7) + 2(18/7) = 48/7 + 36/7 = 84/7 = 12` ✅
- 式(2): `(16/7) - 4(18/7) = 16/7 - 72/7 = -56/7 = -8` ✅

#### グラフィック思考実験
- 式(1): 点 `(0, 6)` と `(4, 0)` を結ぶ直線
- 式(2): 点 `(0, 2)` と `(-8, 0)` を結ぶ直線

**交点**`(16/7, 18/7)`は2直線の交点になりますか？

- 式(1): `3(16/7) + 2(18/7) = 12` ✅
- 式(2): `16/7 - 72/7 = -8` ✅

**一致**。

## 例題3：係数比の考察を含む完全解答

### 問題

以下の連立方程式を解け、解の存在条件を考察せよ：

1. `2x + 3y = 7`
2. `kx + 6y = 12`

**k**が何かの値のとき、解が存在しない条件を特定せよ。

### 解答（ステップバイステップ）

#### ステップ1：代入準備
一方の式を変数について解きます。式(1)が簡単です：

```
式(1): y = (7 - 2x)/3
```

#### ステップ2：代入
式(2)に代入：

```
式(2): kx + 6[ (7 - 2x)/3 ] = 12
```

#### ステップ3：計算
```
kx + 2(7 - 2x) = 12
→ kx + 14 - 4x = 12
→ (k - 4)x = -2
```

#### ステップ4：係数比の考察（重要！）

**「k = 4」**のとき：

係数比 `2/k = 3/6` となるため、係数が比 `1/2`になります：

- 式(1)の係数: `2:3`
- 式(2)の係数: `k:6`

**係数比チェック**:

- `k = 4` → 式(2): `4:6 = 2:3` → 係数比が一致
- `7/k` と `12` → 定数項が一致するか？

式(1)×2:

- `2x + 3y = 7`
- 式(1)×2 → `4x + 6y = 14`

式(2): `kx + 6y = 12`

`k = 4` のとき：

```
4x + 6y = 12
```

**矛盾** `14 = 12` が生じます！

つまり **k = 4** のときは、**無限の解** または **解がない** となりますが、定数項が異なるため **解がありません**。

#### ステップ5：k ≠ 4 の場合

係数比が異なれば、解が1つ存在：

```
(k - 4)x = -2
→ x = -2 / (k - 4)
```

**y**を求める：

```
y = (7 - 2x)/3
→ y = (7 - 2(-2 / (k - 4))) / 3
→ y = (7 + 4/(k - 4)) / 3
```

#### ステップ6：結果

- **k ≠ 4**: **1つの解**
- **k = 4**: **解なし**（定数項が不一致）

### 重要な考察点

**「係数比は解の有無を決定する」**

- `a1/a2 ≠ b1/b2`: 解が1つ
- `a1/a2 = b1/b2` かつ `c1/c2 ≠ a1/a2`: **解なし**
- `a1/a2 = b1/b2` かつ `c1/c2 = a1/a2`: **無限の解**

## まとめ

連立方程式は「2つの変数」を「同時に決定」する手法です。

1. **1つの式を変数について解く**（置換）
2. **もう一方の式に代入**
3. **計算を確認（検算）**
4. **係数比で解の存在を確認**

**重要事項**

- 代入前に分母を共通倍数で消す
- 係数比は解の有無を決定
- 検算は必須
- グラフィックスで交点を可視化して確認
