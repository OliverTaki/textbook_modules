## 概念説明

平面図形における合同と相似は、幾何学の基礎を形成する重要な概念です。合同とは、図形が重ねて完全に一致する場合を指します。具体的には、図形全体のすべての部分（辺、角、面積など）がまったく同じである必要があります。合同条件には、三つ角形で用いられる SAS（挟み角）、ASA（挟み辺）、AAS、SSS、HL（斜辺・鋭角）などの証明ルールがあります。

相似とは、図形の形は全く同じで、大きさだけが異なる場合を指します。相似图形は、すべての対応する角が等しく、対応する辺の比が一定である必要があります。相似比を k とすると、対応する辺の長さの比は k : 1、面積の比は k² : 1 となります。

三つ角形の合同条件として最も重要なのは、SAS（挟み角）、ASA（挟み辺）、AAS、SSS の四つです。これらの条件で証明できるかどうかが、合同の証明作業の核心となります。相似条件では、AA（角）、SAS（辺の比が等しいと角）、SSS（辺の三つとも比が等しい）が用いられます。

これらの概念は、複雑な図形問題を解く際にも重要です。合同と相似の性質を活用することで、未知の長さや角度を計算することが可能になります。また、平行線と合わせて用いると、より高度な幾何問題の解決が可能になります。

## 例題

### 例題 1

**問題:** △ABC ≡ △DEF を証明し、BC の長さを求めなさい。

[条件]
点 A, B, C の座標：A(1, 0), B(0, 3), C(4, 0)
点 D, E, F の座標：D(2, 1), E(1, 4), F(π, 1)
```

[解法]
まず、△ABC の各辺の長さを計算します。

AB = √[(1-0)² + (0-3)²] = √[1 + 9] = √10

BC = √[(0-4)² + (3-0)²] = √[16 + 9] = √25 = 5

AC = √[(1-4)² + (0-0)²] = √[9] = 3

次に、△DEF の各辺の長さを計算します。

DE = √[(2-1)² + (1-4)²] = √[1 + 9] = √10

EF = √[(1-π)² + (4-1)²] = √[(1-π)² + 9]

FD = √[(π-2)²] = π-2

AB と DE は等しく、BC と EF は等しく、AC と FD は等しくなります。これより、△ABC ≡ △DEF が証明できます。

よって、BC = 5 です。

### 例題 2

**問題:** 2 点 P, Q があり、△PAB ~ △QAC となることを証明し、相似比を求めなさい。

[条件]
```
△PAB の辺：PA = 6, PB = 8, AB = 10
△QAC の辺：QA = 12, QC = 16, AC = 20
```

[解法]
まず、二つの三角形の辺の比を計算します。

PA : QA = 6 : 12 = 1 : 2

PB : QC = 8 : 16 = 1 : 2

AB : AC = 10 : 20 = 1 : 2

このように、すべての対応する辺の比が一定（1:2）であるため、△PAB ~ △QAC の相似が証明できます。

次に、対応する角が等しいか確認します。

∠PAB = ∠QAC （対頂角）

∠PBA = ∠QCA （同じく相似条件）

∠APB = ∠AQC （同じく相似条件）

これで、AA（角角）の相似条件が満たされていることが確認できます。

よって、△PAB ~ △QAC であり、相似比は 1:2 です。

## 練習問題

### 問題 1

**問題:** つぎの図形において、△ABC ≡ △ADE を証明し、CD の長さを求めなさい。

[条件]
```
∠BAC = 30°, ∠DAE = 30°
AB = AD = 5
AC = AE = 5√3
BC = DE = 10
```

**解答:**
まず、対応する辺の長さを確認します。
AB = AD = 5
AC = AE = 5√3
BC = DE = 10
また、∠BAC = ∠DAE = 30° が与えられています。

SAS（挟み角）の合同条件を確認します。
AB = AD
AC = AE
∠BAC = ∠DAE
これらの条件より、△ABC ≡ △ADE が証明できます。

次に、CD の長さを求めます。

△ACD は等辺三角形の形を成すため、CD は計算によって求められます。

CD = √[AC² + AD² - 2 × AC × AD × cos(30°)]
   = √[(5√3)² + 5² - 2 × 5√3 × 5 × ½]
   = √[75 + 25 - 50]
   = √50
   = 5√2

**採点基準:**
- SAS 条件の各要素を明確に示している
- 計算過程が正しい
- 最終答案が 5√2 で合っている

### 問題 2

**問題:** 平行線と相似を用いて、未知の長さを求めなさい。

[条件]
```
線分 AB と CD が PQ に平行。
AB = 6, CD = 8, PQ = 12
A, B, C, D が PQ 上にあり、PA = PB
```

**解答:**
AB と CD が PQ に平行であることは、△PAB と △PCD が相似なことを意味します。

相似比を考えます。
PA : PC = AB : CD
  = 6 : 8
  = 3 : 4

P, Q 間の長さが 12 なので、A, B を通る直線と C, D を通る直線の位置関係が重要になります。

AB = 6, CD = 8, PQ = 12
相似比を r とすると，r = 6/8 = 3/4
高さを考慮すると，h : (h+12) = 3 : 4

これを解くと，h = 36

したがって、求めた長さは 36 となります。

**採点基準:**
- 平行線と相似の関係が正しく理解されている
- 相似比の計算が正確
- 計算過程で論理的な根拠がある

### 問題 3

**問題:** 直角三角形において相似を活用して面積を求めなさい。

[条件]
```
直角三角形 ABC（∠C = 90°）において、
CD は高さ線、AB の長さは 10、
A における角度は 30°
```

**解答:**
まず、辺の長さを求めます。

∠A = 30°, ∠C = 90° より ∠B = 60°

AB = 10 （斜辺）

BC = AB × sin(30°) = 10 × ½ = 5

AC = AB × cos(30°) = 10 × ½√3 = 5√3

次に、面積を求めます。

面積 = ½ × BC × AC
    = ½ × 5 × 5√3
    = 25√3 / 2

**採点基準:**
- 三角比の用法が適切
- 面積計算の式が正しい
- 最終答案が 25√3/2 で合っている

## 問題生成ルール

以下のルールに従って新しい問題を生成してください：

1. 合同問題を作成する場合：
   - SAS、ASA、AAS、SSS の条件のいずれかを使用する
   - 計算式を省略し、合同条件を満たすよう設定する
  的证明を明確に提示する
   - 合同条件を満たすよう条件を調整する

2. 相似問題を作成する場合：
   - AA、SAS、SSS の条件のいずれかを使用する
   - 対応する辺の比が計算可能であるように設定する
   - 面積比も考慮する（面積比は相似比の平方になります）
   的证明を明確に提示する

3. 練習問題作成の一般的なルール：
   - 条件の数値は整数で統一（√を含まない）
